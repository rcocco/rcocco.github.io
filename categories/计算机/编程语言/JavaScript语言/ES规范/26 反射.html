26 反射

26.1 Reflect对象

Reflect对象：

  • 是固有对象%Reflect%
  • 是全局对象的"Reflect"属性的初始值
  • 是普通对象
  • 具有[[Prototype]]内部槽，且其值为%Object.prototype%
  • 不是函数对象
  • 没有[[Construct]]内部方法，不能使用new运算符作为构造函数调用
  • 没有[[Call]]内部方法，不能作为函数调用

26.1.1 Reflect.apply ( target, thisArgument, argumentsList )

当使用参数target, thisArgument和argumentsList调用apply函数时，执行以下步骤：

  1. 如果IsCallable(target)为false，抛出 TypeError 异常
  2. 令args为? CreateListFromArrayLike(argumentsList)
  3. 执行PrepareForTailCall()
  4. 返回? Call(target, thisArgument, args)

26.1.2 Reflect.construct ( target, argumentsList [ , newTarget ] )

当使用参数target, argumentsList和newTarget调用construct函数时，执行以下步骤：

  1. 如果IsConstructor(target)为false，抛出 TypeError 异常
  2. 如果newTarget不存在，将newTarget设为target
  3. 否则如果IsConstructor(newTarget)为false，抛出 TypeError 异常
  4. 令args为? CreateListFromArrayLike(argumentsList)
  5. 返回? Construct(target, args, newTarget)

26.1.3 Reflect.defineProperty ( target, propertyKey, attributes )

当使用参数target, propertyKey和attributes调用defineProperty函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 令key为? ToPropertyKey(propertyKey)
  3. 令desc为? ToPropertyDescriptor(attributes)
  4. 返回? target.[[DefineOwnProperty]](key, desc)

26.1.4 Reflect.deleteProperty ( target, propertyKey )

当使用参数target, propertyKey调用deleteProperty函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 令key为? ToPropertyKey(propertyKey)
  3. 返回? target.[[Delete]](key)

26.1.5 Reflect.get ( target, propertyKey [ , receiver ] )

当使用参数target, propertyKey和receiver调用get函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 令key为? ToPropertyKey(propertyKey)
  3. 如果receiver不存在，则将receiver设为target
  4. 返回? target.[[Get]](key, receiver)

26.1.6 Reflect.getOwnPropertyDescriptor ( target, propertyKey )

当使用参数target, propertyKey调用getOwnPropertyDescriptor函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 令key为? ToPropertyKey(propertyKey)
  3. 令desc为? target.[[GetOwnProperty]](key)
  4. 返回FromPropertyDescriptor(desc)

26.1.7 Reflect.getPrototypeOf ( target )

当使用参数target调用getPrototypeOf函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 返回? target.[[GetPrototypeOf]]()

26.1.8 Reflect.has ( target, propertyKey )

当使用参数target, propertyKey调用has函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 令key为? ToPropertyKey(propertyKey)
  3. 返回? target.[[HasProperty]](key)

26.1.9 Reflect.isExtensible ( target )

当使用参数target调用isExtensible函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 返回? target.[[IsExtensible]]()

26.1.10 Reflect.ownKeys ( target )

当使用参数target调用ownKeys函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 令keys为? target.[[OwnPropertyKeys]]()
  3. 返回CreateArrayFromList(keys)

26.1.11 Reflect.preventExtensions ( target )

当使用参数target调用preventExtensions函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 返回? target.[[PreventExtensions]]()

26.1.12 Reflect.set ( target, propertyKey, V [ , receiver ] )

当使用参数target, propertyKey, V和receiver调用set函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 令key为? ToPropertyKey(propertyKey)
  3. 如果receiver不存在，则将receiver设为target
  4. 返回? target.[[Set]](key, V, receiver)

26.1.13 Reflect.setPrototypeOf ( target, proto )

当使用参数target, proto调用setPrototypeOf函数时，执行以下步骤：

  1. 如果Type(target)不是Object，抛出 TypeError 异常
  2. 如果Type(proto)不是Object，或者proto不为null，抛出 TypeError 异常
  3. 返回? target.[[SetPrototypeOf]](proto)

26.2 Proxy对象

26.2.1 Proxy构造函数

Proxy 构造函数：

  • 是固有对象%Proxy%
  • 是全局对象的"Proxy"属性的初始值
  • 作为构造函数调用时创建并初始化一个新的代理奇异对象
  • 不能作为函数调用，以这种形式调用时会抛出异常

26.2.1.1 Proxy ( target, handler )

当使用参数target和handler调用 Proxy 时，执行以下步骤：

  1. 如果NewTarget为undefined，抛出 TypeError 异常
  2. 返回? ProxyCreate(target, handler)

26.2.2.1 Proxy.revocable ( target, handler )

Proxy.revocable函数用于创建可撤销的代理对象。使用参数target和handler调用Proxy.revocable时，执行以下步骤：

  1. 令p为? ProxyCreate(target, handler)
  2. 令steps为Proxy Revocation Functions中定义的算法步骤
  3. 令revoker为! CreateBuiltinFunction(steps, « [[RevocableProxy]] »)
  4. 将revoker.[[RevocableProxy]]设为p
  5. 令result为OrdinaryObjectCreate(%Object.prototype%)
  6. 执行! CreateDataPropertyOrThrow(result, "proxy", p)
  7. 执行! CreateDataPropertyOrThrow(result, "revoke", revoker)
  8. 返回result
26.2.2.1.1 Proxy Revocation Functions

Proxy revocation function是一个匿名内置函数，它能够使一个特定的代理对象失效。

每个Proxy revocation function都有[[RevocableProxy]]内部槽。

当调用Proxy revocation function时，执行以下步骤：

  1. 令F为激活的函数对象
  2. 令p为F.[[RevocableProxy]]
  3. 如果p为null，返回undefined
  4. 将F.[[RevocableProxy]]设为null
  5. 断言：p是代理对象
  6. 将p.[[ProxyTarget]]设为null
  7. 将p.[[ProxyHandler]]设为null
  8. 返回undefined

Proxy revocation function的"length"属性为0
