22 索引集合

22.1 Array对象

数组对象是奇异对象，它可以对某些类的属性名进行特殊处理。 有关特殊处理的定义，请参见9.4.2。

22.1.3 Array原型对象的属性

22.1.3.1 Array.prototype.concat ( …arguments )

当使用零个或多个参数调用concat方法时，它将返回一个数组，该数组包含对象(this)的数组元素，后面跟着依次排列的每个参数的数组元素。

采取以下步骤：

  1. 令O为ToObject(this值)
  2. ReturnIfAbrupt(O)
  3. 令A为ArraySpeciesCreate(O, 0)
  4. ReturnIfAbrupt(A)
  5. 令n为0
  6. 令items为一个列表，它的第一个元素是O，之后的元素从左到右依次是传递给此函数调用的参数。
  7. 只要items不为空，就重复执行：
    1. 从items中删除第一个元素，并令E为此元素的值
    2. 令spreadable为IsConcatSpreadable(E)
    3. ReturnIfAbrupt(spreadable)
    4. 如果spreadable为true，则
      1. 令k为0
      2. 令len为ToLength(Get(E, "length"))
      3. ReturnIfAbrupt(len)
      4. 如果n + len > 2^53-1，抛出TypeError异常
      5. 只要k < len，就重复执行：
        1. 令P为ToString(k)
        2. 令exists为HasProperty(E, P)
        3. ReturnIfAbrupt(exists)
        4. 如果exists为true，则
          1. 令subElement为Get(E, P)
          2. ReturnIfAbrupt(subElement)
          3. 令status为CreateDataPropertyOrThrow (A, ToString(n), subElement)
          4. ReturnIfAbrupt(status)
        5. 将n加1
        6. 将k加1
    5. 否则，E被作为单个元素添加，而不是被展开后再添加
      1. 如果n≥2^53-1，抛出TypeError异常
      2. 令status为CreateDataPropertyOrThrow (A, ToString(n), E)
      3. ReturnIfAbrupt(status)
      4. 将n加1
  8. 令setStatus为Set(A, "length", n, true)
  9. ReturnIfAbrupt(setStatus)
  10. 返回A

concat方法的length属性为1

注：必须在步骤8中显式设置length属性，以确保结果数组不存在尾随元素的情况下，其值正确。

注：concat函数是有意通用的； 它不需要其this值是Array对象。 因此，可以将其转移到其他类型的对象中作为方法。

22.1.3.1.1 运行时语义: IsConcatSpreadable ( O )

参数为O的抽象操作IsConcatSpread执行以下步骤：

  1. 如果Type(O)不是Object，返回false
  2. 令spreadable为Get(O, @@isConcatSpreadable)
  3. ReturnIfAbrupt(spreadable)
  4. 如果spreadable不为undefined，返回ToBoolean(spreadable)
  5. 返回IsArray(O)
