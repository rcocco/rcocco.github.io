20.1 Number对象

20.1.1 Number构造函数

Number构造函数是固有对象％Number％以及全局对象的Number属性的初始值。 当作为构造函数调用时，它将创建并初始化一个新的Number对象。 当Number作为函数而不是构造函数调用时，它将执行类型转换。

Number构造函数被设计为可子类化的。 它可以用作类定义的extends子句的值。 打算继承指定Number行为的子类构造函数必须包括对Number构造函数的super调用，以使用[[NumberData]]内部插槽创建和初始化子类实例。

20.1.1.1 Number ( [ value ] )

当使用参数value调用Number时，将执行以下步骤：

  1. 如果没有传递参数，令n为+0
  2. 否则，令n为ToNumber(value)
  3. ReturnIfAbrupt(n)
  4. 如果NewTarget为undefined，返回n
  5. 令O为OrdinaryCreateFromConstructor(NewTarget, "%NumberPrototype%", «[[NumberData]]» )
  6. ReturnIfAbrupt(O)
  7. 将O的[[NumberData]]内部插槽的值设为n
  8. 返回O
