18 全局对象

唯一的全局对象是在控制流进入任何执行上下文之前创建的。

全局对象没有[[Construct]]内部方法，不能在new运算符中将全局对象作为构造函数。

全局对象没有[[Call]]内部方法，不能将全局对象作为函数调用。

全局对象的[[Prototype]]内部插槽的值取决于实现。

除了本规范中定义的属性之外，全局对象还可以具有其他宿主定义的属性。 这可能包括一个值为全局对象本身的属性。 例如，在HTML文档对象模型中，全局对象的window属性是全局对象本身。

18.1 全局对象的值属性

18.1.1 Infinity

Infinity的值为+∞（参见 6.1.6）。此属性具有特性{ [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: false }

18.1.2 NaN

NaN的值为NaN（参见 6.1.6）。此属性具有特性{ [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: false }

18.1.3 undefined

undefined的值为undefined（参见 6.1.1）。此属性具有特性{ [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: false }

18.2.2 isFinite (number)

isFinite函数是固有对象％isFinite％。 当使用一个参数x调用isFinite函数时，将执行以下步骤：

  1. 令num为ToNumber(number)
  2. ReturnIfAbrupt(num)
  3. 如果num为NaN或+∞或−∞，返回false
  4. 否则，返回true

18.2.3 isNaN (number)

isNaN函数是固有对象％isNaN％。 当使用一个参数x调用isFinite函数时，将执行以下步骤：

  1. 令num为ToNumber(number)
  2. ReturnIfAbrupt(num)
  3. 如果num为NaN，返回true
  4. 否则，返回false

注意：ECMAScript代码测试值X是否为NaN的可靠方法是X！== X形式的表达式。当且仅当X为NaN时，结果为true。

18.2.4 parseFloat (string)

parseFloat函数产生一个Number值，该值由将string参数的内容解释为十进制字面值的结果确定。

parseFloat函数是固有对象％parseFloat％，当使用一个参数string调用parseFloat函数时，将执行以下步骤：

  1. 令inputString为ToString(string)
  2. ReturnIfAbrupt(inputString)
  3. 令trimmedString为inputString的子串，该子串由不是 StrWhiteSpaceChar 的最左边的代码单元和该代码单元右边的所有代码单元组成（换句话说，删除前导空白符）如果inputString不包含任何此类代码单元，则令trimmedString为空字符串。
  4. 如果trimmedString和它的任何前缀都不满足 StrDecimalLiteral 的语法（请参见7.1.3.1），则返回NaN。
  5. 令numberString为trimmedString的最长前缀，该前缀满足 StrDecimalLiteral 语法。最长前缀也可能是trimmedString自己。
  6. 令mathFloat为numberString的数学值(MV)
  7. 如果mathFloat=0，则：
    1. 如果trimmedString的第一个代码单元为-，返回-0
    2. 返回+0
  8. 返回mathFloat的Number值

parseFloat只能将字符串的开头部分解释为Number值； 它忽略了不能解释为十进制字面值表示法的一部分的任何代码单元，并且没有给出任何此类代码单元被忽略的指示。

18.2.5 parseInt (string , radix)

parseInt函数产生一个整数值，该整数值由根据指定的radix来解释字符串参数的结果决定。 string中的前导空白将被忽略。 如果radix为undefined或0，则假定为10，除非数字以代码单元对0x或0X开头（在这种情况下，基数为16）。 如果基数为16，则数字也可以选择以代码单元对0x或0X开头。

parseInt函数是固有对象％parseInt％，调用parseInt函数时，将执行以下步骤：

  1. 令inputString为ToString(string)
  2. ReturnIfAbrupt(string)
  3. 令S为inputString的新创建的子串，它由不是 StrWhiteSpaceChar 的第一个代码单元以及该代码单元之后的所有代码单元组成。 （换句话说，删除前导空白符）如果inputString不包含任何此类代码单元，则使S为空字符串。
  4. 令sign为1
  5. 如果S不为空且S的第一个代码单元为0x002D(HYPHEN-MINUS)减号，令sign为-1
  6. 如果S不为空且S的第一个代码单元为0x002B(PLUS SIGN)加号或0x002D(HYPHEN-MINUS)减号，删除S的第一个代码单元
  7. 令R为ToInt32(radix)
  8. ReturnIfAbrupt(R)
  9. 令stripPrefix为true
  10. 如果R ≠ 0，则：
    1. 如果R < 2或者R > 36，返回NaN
    2. 如果R ≠ 16，令stripPrefix为false
  11. 否则R = 0，令R = 10
  12. 如果stripPrefix为true，则：
    1. 如果S的长度大于等于2，且S的前两个代码单元为0x或0X，则删除S的前两个代码单元，并且令R = 16
  13. 如果S包含不是进制基数R的数字的代码单元，则令Z为S的子串，它由第一个这样的（不是进制基数R的数字的）代码单元之前的所有代码单元组成； 否则，令Z为S。
  14. 如果Z为空串，返回NaN
  15. 令mathInt为由Z表示的R进制下的数学整数值，使用字母A-Z和a-z表示从10到35的数字值（但是，如果R为10，并且Z包含20个以上有效数字，则根据实现方式的选择，第20位之后的每个有效数字都可以用0代替。如果R不是2、4、8、10、16或32，则mathInt可能是依赖于实现的由Z表示的R进制下的数学整数值的近似）
  16. 如果mathInt为0，则：
    1. 如果sign = −1，返回-0
    2. 返回+0
  17. 令number为mathInt的Number值
  18. 返回sign × number

注意：parseInt只能将字符串的开头部分解释为整数值； 它会忽略无法解释为整数符号的一部分的任何代码单元，并且不会给出任何此类代码单元被忽略的指示。
