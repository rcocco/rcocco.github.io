类型转换为对象

使用与不使用new调用的效果一样。

  • 当v是undefined或null时，返回一个空对象，其隐式引用为Object.prototype。
  • 当v不是undefined或null时，使用ToObject(v)抽象操作将参数v转换为对象。
    • 如果v是Object类型，则该函数简单的返回v。
    • 如果v不是Object类型，则该函数会返回v的包装类型。

添加属性/修改特性

单个自有属性

参数O是要添加属性或修改属性特性的对象。
参数propKey是要添加或修改特性的属性对应的属性键。
参数attrs是用来表示特性的对象，它的属性键是小写的特性名，形如：

attrs中不要求把每个特性都列出来，缺失的特性根据情况会维持原值或者采用默认值。
attrs中如果有get或set，它们的属性值必须是函数，否则抛出异常。
attrs中如果存在get或set，就不能有value或writable。反之亦然，如果有value或writable，就不能有get或set。

  • 当对象O没有自有属性propKey时，defineProperty将为对象O添加自有属性propKey。
    • 如果attrs中有value或writable，又或者既没有value和writable也没有get和set，则添加的自有属性propKey是一个数据属性，其特性和attrs中给出的值相同，对于attrs中缺失的特性，[[Value]]是undefined，[[Writable]]和[[Enumerable]]和[[Configurable]]是false。
    • 否则，attrs中一定有get或set，则添加的自有属性propKey是一个访问器属性，其特性和attrs中给出的值相同，对于attrs中缺失的特性，[[Get]]和[[Set]]是undefined，[[Enumerable]]和[[Configurable]]是false。
  • 当对象O具有自有属性propKey时，defineProperty将为对象O更新自有属性propKey的特性。
    • 如果attrs中既没有value和writable也没有get和set，则用attrs更新propKey属性的特性，缺失的特性维持不变。
    • 如果propKey是一个数据属性：
      • 如果attrs中有value或writable，则用attrs更新propKey属性的特性，缺失的特性维持不变。
      • 如果attrs中有get或set，那么propKey会被转换为访问器属性，其特性和attrs中给出的值相同，对于attrs中缺失的特性，[[Get]]和[[Set]]是undefined，[[Enumerable]]和[[Configurable]]维持不变。
    • 如果propKey是一个访问器属性：
      • 如果attrs中有get或set，则用attrs更新propKey属性的特性，缺失的特性维持不变。
      • 如果attrs中有value或writable，那么propKey会被转换为数据属性，其特性和attrs中给出的值相同，对于attrs中缺失的特性，[[Value]]是undefined，[[Writable]]是false，[[Enumerable]]和[[Configurable]]维持不变。

冻结

通过将[[Extensible]]内部槽设为false，并将每个自有属性的configurable和writable特性改为false，使得对象O不能添加、删除任何自有属性，不能修改属性值与属性的任何特性，也不能修改[[Prototype]]以及[[Extensible]]内部槽的值。
如果O不是对象，则简单的返回O而不作任何处理。

密封

通过将[[Extensible]]内部槽设为false，并将每个自有属性的configurable特性改为false，使得对象O不能添加、删除任何自有属性的特性，不能修改属性的特性（但允许将writable改为false），不能修改[[Prototype]]以及[[Extensible]]内部槽的值。
如果数据属性的writable为true，则允许修改数据属性的值。
如果O不是对象，则简单的返回O而不作任何处理。

获取特性

单个自有属性的特性

使用ToObject内置方法将O转为对象obj，然后返回表示obj的自有属性P的特性的对象。
如果obj不存在属性键为P的自有属性，则返回undefined。
对于数据属性，返回的对象具有value、writable、enumerable、configurable四个属性。
对于访问器属性，返回的对象具有get、set、enumerable、configurable四个属性。

所有自有属性的特性

使用ToObject内置方法将O转为对象，然后返回一个新对象。
该新对象的属性键是O的所有自有属性键，值是O的与此属性对应的特性对象。
如果O没有自有属性，则会返回一个空对象。

获取属性键

字符串属性键

使用ToObject内置方法将O转为对象，然后返回一个包含该对象的所有自有、字符串属性键的数组。
数组中，先是按照数字升序排列的整数索引属性键，再是按照创建顺序排列的其他字符串属性键。例如{ c:null, 5:null, a:null, 2:null, [-1]:null, b:null }将返回['2', '5', 'c', 'a', '-1', 'b']。
注意，“整数索引”是用作属性键的字符串，它是规范的数字字符串，即满足ToString(ToNumber(P)) === P的数字字符串，其数字值为[+0, 2^53-1]之间的整数。因此-1并不是整数索引，它属于【按照创建顺序排列的其他字符串属性键】。
