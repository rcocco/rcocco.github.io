生成器函数

概述

生成器函数是一种特殊的函数，生成器函数代码中的yield表达式可以暂停执行函数，并为外部代码（使得生成器函数恢复执行的代码）返回一个值，被暂停的生成器函数也可以在之后恢复执行。在恢复执行生成器函数时，外部代码可以修改导致暂停的yield表达式的值。
生成器函数返回一个生成器对象，一个生成器对象表示对生成器函数代码的一次执行。

创建生成器函数

通过生成器函数定义语法

生成器函数定义语法执行下列过程：

  • 如果有标识符：
    1. 创建一个原型为%Generator%的生成器函数对象F，将F.[[Environment]]设为运行中的执行上下文的词法环境（%Generator%的原型是Function.prototype，%Generator%虽然不是函数对象，却有一个名为prototype的属性，其属性值是一个称为%GeneratorPrototype%的对象）
    2. 注意：生成器函数没有[[Construct]]内部槽，不能作为构造函数调用
    3. 为生成器函数对象F添加prototype属性，属性值是一个原型为%GeneratorPrototype%的对象（%GeneratorPrototype%实现了Iterator接口，原型是%IteratorPrototype%，因此%GeneratorPrototype%也继承了Iterable接口）
    4. 在当前VariableEnvironment词法环境创建一个名为标识符的var变量，其值为函数对象F
  • 如果有可选的标识符：
    1. 创建一个新词法环境funcEnv，其外部词法环境为运行中的执行上下文的词法环境
    2. 创建一个原型为%Generator%的生成器函数对象F，将F.[[Environment]]设为funcEnv
    3. 注意：生成器函数没有[[Construct]]内部槽，不能作为构造函数调用
    4. 为生成器函数对象F添加prototype属性，属性值是一个原型为%GeneratorPrototype%的对象
    5. 在词法环境funcEnv中创建const变量可选的标识符，将其初始化为函数对象F
    6. 返回生成器函数对象F
  • 如果没有可选的标识符：
    1. 创建一个原型为%Generator%的生成器函数对象F，将F.[[Environment]]设为运行中的执行上下文的词法环境
    2. 注意：生成器函数没有[[Construct]]内部槽，不能作为构造函数调用
    3. 为生成器函数对象F添加prototype属性，属性值是一个原型为%GeneratorPrototype%的对象
    4. 返回函数对象F，

调用GeneratorFunction函数会创建一个函数对象，执行过程如下：

  1. 将每个参数使用ToString抽象操作转为字符串
  2. 获取要创建的函数的形式参数列表argList：
    • 如果GeneratorFunction的参数个数为0或1，则argList为空形式参数列表()
    • 否则，GeneratorFunction的参数个数大于等于2，将除了最后一个参数以外的参数使用逗号连接起来，组成形式参数列表(argList)
  3. 获取要创建的函数的函数体body：
    • 如果GeneratorFunction的参数个数为0，则body为空函数体{}
    • 否则，GeneratorFunction的参数个数大于等于1，最后一个参数就是函数体的内容，即{最后一个参数}
  4. 创建一个原型为%Generator%的函数对象F，其形式参数列表为argList，函数体为body，将F.[[Environment]]设为全局词法环境
  5. 注意：生成器函数没有[[Construct]]内部槽，不能作为构造函数调用
  6. 为函数对象F添加prototype属性，属性值是一个原型为%GeneratorPrototype%的对象
  7. 返回F

暂停或结束

与普通函数代码相比，生成器函数代码中有三种yield表达式具有特殊作用：

  • 在执行到yield时，会暂停执行生成器函数，并将生成器的状态修改为suspendedYield，然后为外部代码返回IteratorResult对象{value: undefined, done:false}。
  • 在执行到yield 表达式时，会暂停执行生成器函数，并将生成器的状态修改为suspendedYield，然后为外部代码返回IteratorResult对象{value: 表达式值, done:false}。
  • 在执行到yield * 可迭代对象时，首先获取可迭代对象的迭代器iterator，调用iterator.next(undefined)方法得到IteratorResult对象。如果IteratorResult对象的done为true则表达式的计算值为IteratorResult对象的value，然后继续运行生成器函数。如果done为false，则暂停执行生成器函数，并将生成器状态修改为suspendedYield，然后为外部代码返回IteratorResult对象。之后不断重复下面的流程，直到得到表达式的计算值，又或者结束执行生成器函数：
    • 如果外部代码调用生成器对象的next(val)，则调用iterator.next(val)得到IteratorResult对象。
      • 如果IteratorResult对象的done为true则表达式的计算值为IteratorResult对象的value，然后继续运行生成器函数。
      • 如果IteratorResult对象的done为false，则暂停执行生成器函数，并将生成器状态修改为suspendedYield，然后为外部代码返回IteratorResult对象。
    • 如果外部代码调用生成器对象的return(val)：
      • 如果没有iterator.return方法，则结束执行生成器函数，并将生成器的状态修改为completed，然后为外部代码返回{value: val, done:true}。
      • 如果有iterator.return方法，则调用iterator.return(val)得到IteratorResult对象，如果done为true则会结束执行生成器函数，并将生成器的状态修改为completed，然后为外部代码返回IteratorResult对象。否则，暂停执行生成器函数，并将生成器状态修改为suspendedYield，然后为外部代码返回IteratorResult对象。
    • 如果外部代码调用生成器对象的throw(err)：
      • 如果没有iterator.throw方法，则结束迭代器iterator，然后为yield *表达式抛出异常
      • 如果有iterator.throw方法，则调用iterator.throw(err)，如果iterator.throw(err)抛出了异常，则为yield *表达式抛出异常。如果iterator.throw(err)返回了IteratorResult对象，则若IteratorResult对象的done为true则表达式的计算值为value。否则IteratorResult对象的done为false，暂停执行生成器函数，并将生成器状态修改为suspendedYield，然后为外部代码返回IteratorResult对象。
  • 在执行到return 表达式语句时（包括执行完函数后隐式的return undefined），会结束执行生成器函数，并将生成器的状态修改为completed，然后为外部代码返回IteratorResult对象{value: 表达式值, done:true}
  • 在执行到throw语句时，如果异常没有处理，则会结束执行生成器函数，并将生成器的状态修改为completed，然后为外部代码抛出异常。如果异常处理了，则继续执行生成器函数。

生成器对象

生成器对象是调用生成器函数（不能通过new调用）返回的对象，一个生成器对象表示对生成器函数的一次执行过程，同一个生成器函数可以同时创建多个生成器对象，表示对生成器函数的多次执行过程，每一个生成器对象都是独立的。但生成器函数的闭包只有一个，因此所有生成器对象（执行过程）共享相同的闭包。
生成器对象的原型，就是生成器函数的prototype属性对应的对象。

内部属性

生成器对象具有两个内部属性：

  • GeneratorState：指示生成器的当前执行状态，可能的值有suspendedStart（尚未执行过）, suspendedYield（已暂停执行）, executing（正在执行）和completed（已执行完）
  • GeneratorContext：在执行该生成器的代码时使用的执行上下文

每个新创建的生成器对象最初的状态都是suspendedStart。

方法

生成器对象实现了Iterable和Iterator接口，具有next、return和throw方法，调用这三个方法可以使生成器函数继续执行或者结束生成器函数的执行过程。

  • 调用生成器对象的next(val)：
    • 如果生成器的状态是completed，则返回{value: undefined, done: true}
    • 如果生成器状态是suspendedStart，则恢复执行生成器函数，然后返回生成器函数代码所返回的值。
    • 否则（此时状态是suspendedYield），为暂停生成器函数的yield表达式（如果存在）赋予val值，然后恢复执行生成器函数，最后返回生成器函数代码所返回的值。
  • 调用生成器对象的return(val)：
    • 将生成器的状态修改为completed，使得生成器函数的执行过程结束，然后返回{value: val, done:true}。
  • 调用生成器对象的throw(err)：
    • 如果生成器状态是suspendedStart或completed，则结束执行生成器函数，并将生成器的状态修改为completed。最后throw函数将抛出异常err
    • 否则（此时状态是suspendedYield），在暂停生成器函数的yield表达式的位置，抛出异常err，然后恢复执行生成器函数，最后返回生成器函数代码所返回的值。
