process模块

process模块提供了有关当前 Node.js 进程的信息并对其进行控制。
process是一个全局变量，因此不需要显式的require。
但如果非要require也是可以的。

命令行参数

process.argv属性返回一个命令行参数（以空格分隔）数组。
第一个元素是Node执行程序的路径（也可能简写为node）
第二个元素是正在执行的 JavaScript 文件的路径。
从第三个元素开始，是其他命令行函数。
process.argv0返回process.argv[0]的只读副本。

工作目录

cwd()方法返回当前工作目录。
chdir(directory)方法用于改变工作目录，如果操作失败（例如，指定的 directory 不存在）则抛出异常。

退出程序

process.exit()方法用于强制进程尽快退出，即使还有尚未完全完成的异步操作在等待。

当前进程信息

process.pid返回当前进程的PID
process.ppid返回当前进程的父进程的PID
