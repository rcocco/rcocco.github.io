create-react-app
2021-06-15|计算机前端

创建项目：npx create-react-app 应用名(文件夹名)
创建使用TypeScript模板的项目：npx create-react-app 应用名(文件夹名) --template typescript
在开发模式下运行程序：npm start
构建为生产模式：npm run build

默认不包含polyfill，要兼容需要手动包含
https://github.com/facebook/create-react-app/blob/master/packages/react-app-polyfill/README.md

文件结构

public/index.html（页面模板）和src/index.js（JS入口）两个文件必须存在。
其他文件都可以不存在。
可以在src目录下创建子目录，但webpack只会处理在src目录的文件，所以CSS和JS都放在src里不要放在子目录。

选项

package.json文件的browserslist选项指定了输出的JavaScript代码兼容哪些浏览器。

production用于build命令，development用于start命令。
编辑browserslist后，需要删除node_modules/.cache来使其生效。
