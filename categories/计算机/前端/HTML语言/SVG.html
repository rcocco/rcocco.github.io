基本元素

  • <svg> 包裹并定义整个矢量图。<svg> 标签之于矢量图就如同 <html> 标签之于一个 web 页面。
  • <line> 创建一条直线。
  • <polyline> 创建折线。
  • <rect> 创建矩形。
  • <ellipse>创建圆和椭圆。
  • <polygon>创建多边形。
  • <path>通过指定点以及点和点之间的线来创建任意形状。
  • <defs>定义一个可复用的图形。初始情况下<defs>里面的内容是不可见的。<defs>标签之于矢量图就如同<head>标签之于一个 web 页面。
  • <g>将多种形状组合起来。将组合后的形状置于<defs>中可以让它能够被复用。
  • <symbol>类似于一个组合，但是拥有一些额外的特性。通常被置于 <defs>标签中便于复用。
  • <use>获取在<defs>中定义的复用对象并在 SVG 中显示出来。

坐标轴

x表示横轴，正方向是向右的，y表示纵轴，正方向是向下的。
handcodesvg.html绘图板中，一个格子是10px，中等粗细的线之间是100px。

创建容器

<svg>元素是容器，最好指定width和height属性
例如<svg width="750" height="500"></svg>

直线

<line>元素用来画直线，需要指定起点的坐标和终点的坐标，使用4个属性来表示：x1为起点横轴坐标，y1为起点纵轴坐标，x2为终点横轴坐标，y2为终点纵轴坐标。
例如<line x1="3" y1="3" x2="48" y1="3"></line>

折线

<polyline>元素用来画折线，它需要指定构成折线的点的坐标，使用2个以空格分隔的数字表示，多个点之间可以用逗号分隔，也可以用空格或者换行符分隔。
这些点的坐标需要写在<polyline>的points属性中。
例如<polyline points="3 3,30 28,3 53"></polyline>

矩形

<rect>元素用来画矩形，它需要指定矩形左上角的坐标、矩形的宽度和高度。如果有需要也可以指定圆角半径。使用4个属性表示：x为左上角横轴坐标，y为左上角纵轴坐标，width为矩形宽度，height 为矩形高度。（rx和ry圆角半径）
例如<rect x="3" y="3" width="80" height="60"></rect>

椭圆

<ellipse>元素用来画椭圆，它需要指定椭圆中心，以及水平半径和垂直半径。使用4个属性表示：cx为中心横轴坐标，cy为中心纵轴坐标，rx为水平半径，ry为垂直半径。
例如<ellipse cx="43" cy="43" rx="40" ry="40"></ellipse>

多边形

<polygon>元素用来画多边形，它和<polyline>完全相似，只是<polygon>图形是自动闭合的，而<polyline>图形不是闭合的。
它需要指定构成折线的点的坐标，使用2个以空格分隔的数字表示，多个点之间可以用逗号分隔，也可以用空格或者换行符分隔。
这些点的坐标需要写在<polygon>的points属性中。
例如<polygon points="35 23, 60 43, 35 63"></polygon>

路径

<path>元素用来创建路径，类似创建多边形，但你可以在点与点之间创建曲线而不是直线。
d属性用于定义路径的各种操作。
路径的每个操作是由单个字母加坐标组成的。
常用操作如下：
M表示moveto，后面跟一个坐标，大写M表示后面是一个绝对坐标，小写M表示后面是一个相对坐标。意思是把准备绘画的当前位置调整为指定坐标。
L表示lineto，后面跟一个坐标，意思是从当前位置到坐标所指定的新位置画一条线。
Z表示闭合路径。后面不需要跟东西。它会在当前位置与路径起始点之间画一条直线来闭合形状。
例如<path d="M 18 3,L 46 3,L 46 40,L 61 40,L 32 68,L 3 40,L 18 40,Z"></path>
逗号分隔不是必须的，也可以用空格分隔。甚至可以把逗号放在坐标x和y之间，因此实际上路径操作只看操作个数，不看逗号位置。

填充

通过内联样式，可以给图形添加填充，例如<ellipse style="fill:black;" cx="43" cy="65" rx="5" ry="5"></ellipse>
