Element结点
2021-06-15|计算机前端DOM

Element结点

Element结点简称元素。
元素有关联的namespace、namespace prefix、local name、custom element state、custom element definition、is value。元素创建时，所有这些关联属性都被初始化。
custom element state为uncustomized或custom的元素称为定义的。custom element state为custom的元素称为自定义的。
元素有一个关联的影子根，除非另有说明否则为null，当影子根非null时元素被称为影子宿主。

命名空间

  • namespaceURI属性返回元素的命名空间。
  • prefix属性返回元素的命名空间的前缀。

标签名

localName属性返回元素的标签名。

限定名

当命名空间前缀prefix为null时，限定名就是标签名localName。
当命名空间前缀prefix不是null时，限定名是前缀:标签名。
元素的tagName属性返回限定名，但要注意当元素的命名空间是HTML命名空间，且元素的结点文档是HTML文档时，tagName会返回大写的限定名而不是原本的限定名。

ID

每个元素可以有一个唯一标识符（ID）。
id属性的get访问器方法会获取元素的特性列表中id特性的值，如果不存在则返回空串。
id属性的set访问器方法会修改元素的特性列表中id特性的值（如果存在），或是向特性列表中添加id特性（如果不存在）

slot

slot属性的get访问器方法会获取元素的特性列表中slot特性的值，如果不存在则返回空串。
slot属性的set访问器方法会修改元素的特性列表中slot特性的值（如果存在），或是向特性列表中添加slot特性（如果不存在）

获取关联元素

子元素

  • children属性返回一个动态的HTMLCollection集合，匹配this的子结点中类型为Element的结点。
  • firstElementChild属性返回this的子结点中第一个类型为Element的结点，如果不存在则返回null
  • lastElementChild属性返回this的子结点中最后一个类型为Element的结点，如果不存在则返回null
  • childElementCount属性返回this的子结点中，类型为Element的结点的数量。

兄弟元素

  • previousElementSibling方法返回当前元素的前面的兄弟结点中最近的Element结点，如果不存在则返回null。
  • nextElementSibling方法返回当前元素的后面的兄弟结点中最近的Element结点，如果不存在则返回null。

类名

每个元素可以有一个类名(class)特性，类名特性的值是使用空白符([\t\n\f\r ])分隔的类名。
可以像其他特性一样直接设置整个类名特性的值，也可以使用classList属性返回的DOMTokenList对象来操作类名特性值中的单个类名，而不必自己去拼接字符串得到完整的类名特性值。
注意，DOMTokenList对象将类名当成一个ordered set来进行处理，重复的会被忽略。尽管设置所有类名时，可能会有重复的类名，但调用classList上的添加/删除/替换/切换类名方法后，所有重复的类名都会被清除。

获取/设置所有类名

className和classList.value属性的get访问器方法，会获取元素的特性列表中class特性的值，如果不存在则返回空串。
className、classList和classList.value属性的set访问器方法会修改元素的特性列表中class特性的值（如果存在），或是向特性列表中添加class特性（如果不存在）

获取单个类名

  • classList.item(index)方法返回第index个类名，如果不存在则返回null。
  • childNodes[index]表达式返回第index个类名，如果不存在则返回undefined。

类名个数

classList.length属性返回类名特性中类名的个数。

类名是否存在

classList.contains方法返回一个布尔值，表示类名token是否存在

添加类名

添加由参数传递的一个或多个类名，需要注意的是：

  • 参数字符串中不能有空串，否则会抛出异常
  • 参数字符串中不能有空白符([\t\n\f\r ])，否则会抛出异常
  • 如果类名中已经存在参数字符串，则这个参数会被忽略。
  • 函数执行完后，class特性中重复的类名会被清除。

删除类名

删除由参数传递的多个类名，需要注意的是：

  • 参数字符串中不能有空串，否则会抛出异常
  • 参数字符串中不能有空白符([\t\n\f\r ])，否则会抛出异常
  • 如果类名中不存在参数字符串，则这个参数会被忽略。
  • 函数执行完后，class特性中重复的类名会被清除。

切换类名存在性

如果类名中存在token，则删除该类名，然后返回false。
如果类名中不存在token，则添加该类名，然后返回true。
返回值表示的是函数执行完后，当前类名中是否存在token。

需要注意的是：

  • 参数字符串中不能有空串，否则会抛出异常
  • 参数字符串中不能有空白符([\t\n\f\r ])，否则会抛出异常
  • 函数执行完后，class特性中重复的类名会被清除。

替换类名

如果类名中不存在token，则什么也不做并返回false。
否则，将类名中的token替换为newToken，然后清除所有重复的类名，最后返回true。

遍历类名

classList属性返回的DOMTokenList是一个可迭代对象，迭代结果是类名。

元素特性

每个元素都有一个特性（结点）列表，最初为空。特性列表可以通过元素的attributes属性返回的NamedNodeMap对象访问与操作，除此之外，元素上也有一些方法用于方便的操作特性。

特性列表是否为空

hasAttributes方法返回一个布尔值，表示元素的特性列表是否为空。

特性的个数

attributes.length属性返回特性列表中特性结点的个数。

全体特性的限定名

getAttributeNames方法返回一个字符串数组，由特性列表中所有特性的限定名组成。字符串数组中可能存在重复。

是否存在特性

  • hasAttribute方法返回一个布尔值，表示特性列表中是否存在限定名为qualifiedName的特性。如果元素的命名空间为HTML命名空间，且是HTML文档，则参数qualifiedName不区分大小写。
  • hasAttributeNS方法返回一个布尔值，表示特性列表中是否存在命名空间为namespace且本地名为localName的特性

获取特性

获取结点

返回特性列表中第一个限定名为qualifiedName的特性结点。
如果元素的命名空间为HTML命名空间，且是HTML文档，则参数qualifiedName不区分大小写。

返回特性列表中第一个命名空间为namespace且本地名为localName的特性结点。
注意参数namespace为空串会被替换为null。

attributes.item(index)方法返回特性列表中第index个特性结点，如果不存在则返回null。
attributes[index]表达式返回特性列表中第index个特性结点，如果不存在则返回undefined。

获取值

  • getAttribute方法返回特性列表中第一个限定名为qualifiedName的特性。如果不存在这样的特性，则返回null。如果元素的命名空间为HTML命名空间，且是HTML文档，则参数qualifiedName不区分大小写。
  • getAttributeNS方法返回特性列表中第一个命名空间为namespace，本地名为localName的特性。如果不存在这样的特性，则返回null。

也可以根据相同参数获取特性结点后再获取值，但相对麻烦。

添加/修改特性

给定结点

  • 如果特性列表中存在命名空间和本地名与参数attr结点相同的结点，则将第一个这样的结点替换为参数attr，然后将被替换的结点的所属元素设为null，最后返回被替换的结点。
  • 否则，向特性列表中添加参数attr结点，然后返回null。

注意参数attr结点的所属元素必须要么是null，要么是当前元素，否则会抛出异常。

给定值

  • 如果特性列表中存在命名空间为namespace、限定名为qualifiedName的，则将第一个这样的结点的值替换为参数value。如果元素的命名空间为HTML命名空间，且是HTML文档，则参数qualifiedName不区分大小写。
  • 否则，创建一个命名空间、限定名和值为给定参数的特性结点，然后向特性列表中添加该结点。

删除特性结点

给定结点

如果特性列表中存在参数attr结点，则删除它并返回。
如果特性列表中不存在参数attr，则抛出异常。

给定名字

使用给定的参数调用getNamedItem或getNamedItemNS方法来查找特性结点。
如果找到了则删除并返回该结点，如果没找到则抛出异常。

使用给定的参数调用getNamedItem或getNamedItemNS方法来查找特性结点。
如果找到了则删除，如果没找到什么也不做。

这两组函数的区别就是是否返回被删除的结点，以及是否抛出异常。

切换特性存在性

如果特性列表中存在限定名为qualifiedName的特性，则删除该特性，然后返回false。
如果特性列表中不存在限定名为qualifiedName的特性，则添加该特性，且值为空串，然后返回true。
返回值表示的是函数执行完后，当前特性列表中是否存在限定名为qualifiedName的特性。

需要注意的是，如果元素的命名空间为HTML命名空间，且是HTML文档，则参数qualifiedName不区分大小写。

影子根

每个元素具有一个影子根属性，其值为null或一个ShadowRoot对象。该ShadowRoot对应的影子DOM的宿主为当前元素。

获取影子根

shadowRoot属性用于获取影子根。
如果元素不存在影子根，或者存在一个模式为closed的影子根，则返回null。
否则返回该影子根。

创建ShadowRoot

此方法将会创建一个ShadowRoot对象，将其设置为元素的影子根，然后返回该对象。

匹配选择器

当前元素是否匹配选择器

matches方法返回一个布尔值，表示在当前元素的根上进行选择器匹配，能否匹配到当前元素。

最近的匹配选择器的元素

closest方法返回当前元素，或者它的祖先元素中，离它最近的匹配选择器的元素。如果没有这样的元素，则返回null。

查找后代元素结点

根据标签名查找

返回一个动态的HTMLCollection集合。

  • getElementsByTagName：当参数qualifiedName为"*"时，集合匹配的是this后代结点中所有的Element结点。
    • 对于通常的HTML文档，集合的根为this，过滤器为匹配this的后代结点中：
      • 位于HTML命名空间，标签名为qualifiedName（不区分大小写）的Element结点。
      • 不在HTML命名空间，标签名为qualifiedName（区分大小写）的Element结点。
    • 例如getElementsByTagName('SVG')就获取不到元素，因为不在HTML命名空间，标签名为svg在区分大小写的情况下不等于参数SVG。
  • getElementsByTagNameNS：当参数namespace和localName都为"*"时，集合匹配的是this后代结点中所有的Element结点。
    • 如果参数namespace不为"*"，则匹配的结点要求namespaceURI等于namespace
    • 如果参数localName不为"*"，则匹配的结点要求localName属性等于参数localName

由于大多数情况都不需要考虑命名空间，因此使用getElementsByTagName更合适。

根据类名查找

返回一个动态的HTMLCollection集合。
参数classNames会根据空白符（[\t\n\f\r ]+）切分成类名，集合会匹配this后代结点中，具有classNames中全部类名的Element结点。
也就是集合中的结点会包含所有类名，它可以有比classNames更多的类名，但不能缺少className中的类名。
quirk（怪异模式）的文档在类名匹配时不区分大小写，标准模式的文档进行类名匹配时区分大小写。

插入结点

插入元素

  • 当where为beforebegin时：
    • 若当前元素没有父结点，则返回null
    • 若当前元素有父结点，则将参数element插入到当前元素前面，返回参数element
  • 当where为afterend时：
    • 若当前元素没有父结点，则返回null
    • 若当前元素有父结点，则将参数element插入到当前元素后面，返回参数element
  • 当where为afterbegin时：将参数element插入到当前元素的子结点的开头，返回参数element
  • 当where为beforeend时：将参数element插入到当前元素的子结点的末尾，返回参数element

插入文本

  • 当where为beforebegin时：
    • 若当前元素没有父结点，则返回null
    • 若当前元素有父结点，则创建一个值为data的Text结点插入到当前元素前面，返回参数element
  • 当where为afterend时：
    • 若当前元素没有父结点，则返回null
    • 若当前元素有父结点，则创建一个值为data的Text结点插入到当前元素后面，返回参数element
  • 当where为afterbegin时：创建一个值为data的Text结点插入到当前元素的子结点的开头，返回参数element
  • 当where为beforeend时：创建一个值为data的Text结点插入到当前元素的子结点的末尾，返回参数element

子结点最前面

prepend方法会将参数nodes中的所有结点插入到this子结点的最前面，注意参数中的字符串会被替换为对应的文本结点。

子结点最后面

append方法会将参数nodes中的所有结点插入到this子结点的最后面，注意参数中的字符串会被替换为对应的文本结点。

当前结点前面

before方法会将参数nodes中的所有结点作为兄弟结点插入到this前面（准确的说是前面第一个不在nodes中的结点之后的位置），注意参数中的字符串会被替换为对应的文本结点。

当前结点后面

after方法会将参数nodes中的所有结点作为兄弟结点插入到this后面（准确的说是后面第一个不在nodes中的结点之前的位置），注意参数中的字符串会被替换为对应的文本结点。

替换结点

替换子结点

replaceChildren方法会移除this的所有子结点，然后将参数nodes中的所有结点插入到this子结点中，注意参数中的字符串会被替换为对应的文本结点。

替换当前结点

replaceWith方法会用参数nodes中的所有结点替换this。注意参数中的字符串会被替换为对应的文本结点。

删除当前结点

remove()方法会从this的父结点的孩子中删除当前结点。

Attr结点

Attr结点简称特性。
由于历史原因，特性具有namespace、namespace prefix、local name、value、element。如果在今天重新设计，特性会只有name和value。

A特性是指一个Attr对象，其localName为A，namespaceURI和prefix为null。

命名空间

  • namespaceURI属性返回特性的命名空间。
  • prefix属性返回特性的命名空间的前缀。

标签名

localName属性返回特性的名字。

限定名

当命名空间前缀prefix为null时，限定名就是标签名localName。
当命名空间前缀prefix不是null时，限定名是前缀:标签名。
特性的name属性返回限定名。

值

  • value属性的get访问器方法可以获取特性的值。
  • value属性的set访问器方法可以设置特性的值。

所属元素

ownerElement属性返回特性的所属元素，如果不存在则返回null。
