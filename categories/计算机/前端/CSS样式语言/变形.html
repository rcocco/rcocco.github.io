变形属性

transform属性使你可以旋转、缩放、倾斜、平移元素。

  • 取值语法：
  • 初始值：none
  • 适用于：可变形元素（除原子级行内框之外的所有元素）
  • 计算值：指定的值，相对长度会变为绝对长度
  • 百分数：相对于范围框（一般boreder-box）
  • 继承性：否
  • 动画性：是

原子级行内框指<span>和<a>等元素，它们可以随块级父元素一起变形，但不能直接变形。因为一个元素的行框很可能有多行，无法确定是作为整体变形还是单独每行变形。
CSS中2D变换的坐标轴跟平常的直角坐标系略有不同，x轴和直角坐标系一样，但y轴的正向是朝下的。
在3D变换的坐标轴中，z轴上的值越大，越靠近屏幕，z值越小越远，类似z-index。

在变换过程中，各轴也随之变换，因此旋转元素后，坐标轴也会旋转。
变形后元素有自己的堆叠上下文，在页面上所占的空间与变形前保持不变。
变形函数从左到右依次执行，因此顺序不同结果很可能也不同。

对于多个变形函数例如transform: rotate(30deg) skewX(-25deg) scaleY(2)，如果其中有任何一个函数是无效的，整个属性都会被忽略，所有的变形函数都不会执行。
多个transform不叠加，后面的规则覆盖前面的规则。

平移函数

平移函数的值可以是长度值<length>也可以是百分数值<percentage>，百分数值是相对于元素自身的尺寸计算的，X轴的移动量相对于宽度，Y轴移动量相对于高度。

  • 沿X轴移动：translateX(<length-percentage>)
  • 沿Y轴移动：translateY(<length-percentage>)
  • 2D平移：translate(<length-percentage> [, <length-percentage> ]?)
    第一个参数是沿X轴的移动量，正值向右，负值向左。
    第二个可选参数是沿Y轴的移动量，如果省略第二个参数，则默认为0，即Y轴方向不移动。正值向下，负值向上。

  • 沿Z轴移动：translateZ(<length>)
  • 3D平移：translate3d(<length-percentage>,<length-percentage>,<length>)
    3D平移的参数必须是三个，少于三个是无效的。

缩放函数

缩放函数的值是一个数字，表示变换后的元素是原来元素的倍数。scaleX(2)表示宽度是原来的2倍。缩放后，坐标轴的单位也会缩放，translateX(10px)此时会移动20px。

  • X方向缩放：scaleX(<number>)
  • Y方向缩放：scaleY(<number>)
  • 2D缩放：scale(<number> [, <number>]?)
    如果只提供一个参数，则X方向和Y方向都使用该值。

  • Z方向缩放：scaleZ(<number>)
  • 3D缩放：scale3d(<number>,<number>,<number>)
    3D缩放的参数必须是三个，少于三个是无效的。

旋转函数


旋转函数绕某个轴旋转元素，或者绕3D空间中的一个向量旋转元素。它接受角度值<angle>

  • rotate(<angle>)：2D旋转，等同于rotateZ，绕Z轴旋转XoY平面。右手大拇指指向Z轴负无穷远处，四指弯曲方向就是旋转方向。
  • rotateX(<angle>)：绕X轴旋转YoZ平面，右手大拇指指向X轴负无穷远处，四指弯曲方向就是旋转方向。
  • rotateY(<angle>)：绕Y轴旋转XoZ平面，右手大拇指指向Y轴负无穷远处，四指弯曲方向就是旋转方向。
  • rotateZ(<angle>)


rotate3d(<number>,<number>,<number>,<angle>)：3D旋转，前三个数字指定空间向量在X、Y、Z轴上的分量，第四个值是角度值，指定绕向量旋转的量。右手大拇指指向向量的反方向，四指弯曲方向就是旋转方向。

倾斜函数

  • skewX(<angle>)：沿X轴倾斜
  • skewY(<angle>)：沿Y轴倾斜
  • skew(<angle>[,<angle>]?)：通过矩阵运算[ax,ay]实施2D倾斜。和同时应用skewX、skewY的结果不同。第一个角度值是x轴倾斜角度，第二个角度值是y轴倾斜角度。


透视函数

perspective(<length>)函数改变透视锥体的距离（观察点与元素平面的距离），越小的值越靠近元素，越大的值越远离元素，不设置透视时，观察点在无穷远处。
函数值必须是正数，而且不能是零，否则将导致函数被忽略。
只应用该函数不会有任何视觉效果，它只对它后面的变形函数起作用。

矩阵函数

matrix(<number>[,<number>]{5,5})：2D空间矩阵变换。
函数值是六个以逗号分隔的数字，不能多也不能少。数字可正可负。
描述的是元素变形后的最终形态，因此用矩阵函数进行动画时，只会以最短路径转变过去，而不会像其他变形函数一样旋转若干圈。

matrix3d(<number>[,<number>]{15,15})：3D空间矩阵变换。
函数值是16个以逗号分隔的数字。

关于渐变

如果想看见每一个变换函数的过渡效果，过渡前后必须变换函数位置必须对应，否则它会以最短路径改变。

变形原点

默认的变形原点是元素的中心，通过transform-origin可以修改原点

  • 取值语法：
  • 初始值：50% 50% 0
  • 适用于：可变形元素（除原子级行内框之外的所有元素）
  • 计算值：对于百分数，指定的值。相对长度会变为绝对长度。
  • 百分数：相对于范围框（一般boreder-box）
  • 继承性：否
  • 动画性：是

1值语法：

  • <length-percentage>、left、center、right、top、bottom：表示水平偏移（距离左边），垂直偏移为初始值50%，Z轴偏移为初始值0。

2值语法：
第一个值为水平偏移，不能是关键字top和bottom。第二个值为垂直偏移（距离上边），不能是关键字left和right。Z轴偏移为初始值0。

3值语法：
前两个值和2值语法相同。第三个值为Z轴偏移，只能是长度值。


选择3D变形方式

使用translate3d()或rotateY()等函数在三维空间中改变元素形态时，子元素默认都是扁平的，即子元素好像是贴在父元素上的一张张纸片，随父元素变形。
transform-style属性可以改变这个行为，令子元素在3D空间中单独存在。

  • 取值语法：
  • 初始值：flat
  • 适用于：可变形元素（除原子级行内框之外的所有元素）
  • 计算值：指定的值
  • 继承性：否
  • 动画性：否

只有当以下属性为以下值时，preserve-3d才有效，否则会强制重置为flat

  • overflow: visible
  • filter: none
  • clip: auto
  • clip-path: none
  • mask-image: none
  • mask-border-source: none
  • mix-blend-mode: normal
  • isolation: ioslate（计算值为它也可）

透视属性

透视深度

perspective属性用于定义透视锥体的深度。

  • 取值语法：
  • 初始值：none
  • 适用于：可变形元素（除原子级行内框之外的所有元素）
  • 计算值：绝对长度或none
  • 继承性：否
  • 动画性：是

该属性会应用到目标元素的所有子元素上，也就是说子元素和它共用同一个透视锥体，如第三行的图。就像把三个照片贴在透明玻璃片上，绕着玻璃片的轴旋转。
如果使用变形透视函数perspective()，每个子元素将有独立的透视锥体，如第二行的图。

透视原点

perspective-origin属性用于定义透视原点

  • 取值语法：
  • 初始值：50% 50%
  • 适用于：可变形元素（除原子级行内框之外的所有元素）
  • 计算值：对于百分数，指定的值。相对长度会变为绝对长度。
  • 百分数：相对于范围框（一般boreder-box）
  • 继承性：否
  • 动画性：是

该属性定义的是视线汇聚于哪一点，例如下面尺子的刻度延长线汇聚于该点。

背面

backface-visibility属性用于设置当元素通过背面朝向用户时，是否显示。
元素背面朝向用户仅在三维空间旋转时才可能发生。

  • 取值语法：
  • 初始值：visible
  • 适用于：可变形元素（除原子级行内框之外的所有元素）
  • 计算值：对于百分数，指定的值。相对长度会变为绝对长度。
  • 继承性：否
  • 动画性：否
