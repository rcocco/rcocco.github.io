值和单位
2020-10-27|计算机前端CSS样式语言

关键字

声明的属性值有时是一个英文单词，这种情况下将值称作关键字。例如none和normal。
每个属性能使用的关键字大多不同。

CSS3定义了三个所有属性都能使用的关键字：

  • inherit：将该元素的属性值设置成跟父元素同一属性的值一样。可以使得不继承的属性强制继承。
  • initial：将该元素的属性值设置成预定义的初始值。某些属性的初始值取决于用户代理，用该关键字可以简单地恢复为用户代理的默认设置。
  • unset：对于继承的属性，等价于inherit。对于不继承的属性，等价于initial。

字符串

声明的属性值有时是放在单引号或双引号内的字符序列，这种情况下将值称为字符串。例如"I like to play with strings"和'Strings are fun to play with.'
双引号字符串里面可以出现单引号，但不能直接出现双引号，必须写成反斜杠\加双引号"，即\"。
单引号字符串里面可以出现双引号，但不能直接出现单引号，必须写成反斜杠\加单引号'，即\'。
如果为了代码看起来可读，想把一个很长的字符串分成多行来写，可以在一行的结尾加上反斜杠\，然后另起一行写。CSS会认为这种拆成多行来写的字符串和在一行来写的字符串完全相同。
字符串不能直接包含换行，为了在字符串中插入换行符，需要在想换行的地方添加Unicode字符\A。

URL

声明的属性值有时是一个URL，引用URL的一般格式形如url(protocol://server/pathname)或url(pathname)，即在url()的括号中写入URL。标识符url与圆括号()之间不能出现空白符。URL可以用单引号或双引号引起来，如果不用引号引起来，那么URL中的任何空格、单引号和双引号前必须加反斜杠\转义。
在CSS中，相对URL相对于所在的样式表，而不是使用样式表的文档。

图像

声明的属性值有时是一个图像，句法用<image>表示。
<image>表示下面三种值中的一个：

  • <url>：图像的URL。
  • <image-set>：一系列图像，根据值中的条件选择，例如image-set()
  • <gradient>：线性渐变或径向渐变图像，可以是单个的，也可以重复。

渐变

渐变是一种图像类型，渐变分为线性渐变和径向渐变，每种渐变又分为循环渐变和不循环渐变。
相关示例：http://projects.verou.me/css3patterns/

梯度线的起点是有一条经过该点，并垂直于梯度线的线，同时这条线与背景定位区域的相交。

更具体的说，梯度线是这样构建的：

  1. 过图像的中点，把梯度线声明象限的那个角的两侧相邻的角连接起来
  2. 梯度线与该连接线垂直，指向声明的象限
  3. 作连接线的平行线确定起点与终点


接着是一系列色标(color stop)和中色点(color hint)，最后又是一个色标。
色标是一个颜色值，后面可以提供位置值，位置值表示该颜色所处的位置，不提供位置颜色将均匀排列。第一个色标如果不提供位置，表示位置在开头。最后一个色标如果不提供位置，表示位置在结尾，如果色标位置超出了范围，渐变也将在范围末尾结束，如果最后一个色标位置小于范围，它将一直延伸到渐变结束。
如果色标的位置值是一个百分数，则它是从起始点到终止点的距离的百分数。
如果色标的位置值是一个长度值，则它是从起始点开始的长度，如果最后一个色标的位置值小于梯度线长度，则最后一个色标的颜色一直延伸到末尾。如果色标的位置值大于梯度线长度，则该色标很可能不会显示出来，因为到终止点的长度就停止了。
如果部分色标有位置，部分没有，没有的色标将均匀分布在有位置的色标之间。
如果色标位置重叠，该位置前后仍然有渐变，但是该位置的颜色将突然变换，两边的渐变色也不一样：

如果后面的色标位置设置的比前面的位置还靠前，那么位置会被替换为前面色标位置中选出最大的位置值。例如#spectrum {background-image: linear-gradient(90deg, red 0%, orange, yellow, green 50%, blue 40%, indigo, violet)};中blue的位置值40%会被替换为前面最大的50%。
linear-gradient(0deg,red,green)表示梯度向量朝上，底部是红色，然后过渡到上方绿色。

中色点是一个百分数只，用来修改两侧色标的混合模式，默认的混合模式是线性的。
百分数值相对于梯度线总长度计算（即背景范围）。

标识符

声明的属性值有时接受标识符值，即用户定义的某种标识符，最常见的有生成的列表序号。在取指句法中用<identifier>表示。标识符本身是词，而且区分大小写。
如果属性值既接受标识符也接受关键字，不能把标识符定义的跟有效关键字一样。

整数

整数是其他类型值的基础，由一个或多个数组成，前面还可以有+号表示正数或-号表示负数。在取值句法中用<integer>表示。例如17、-753等等。
某些属性对整数的取值范围有规定，范围之外的整数值是无效的，会导致整个声明被忽略。

数字

数字值是<integer>或实数。实数指整数后跟一个点，再跟一些整数，即<integer>.<integer>，前面还可以有+号表示正数或-号表示负数。在取值句法中用<number>表示。例如6.626、-1.7588。
某些属性对数字的取值范围有规定，范围之外的数字值是无效的，会导致整个声明被忽略。

百分数

百分数值的形式是<number>%，在取值句法中用<percentage>表示。例如50%、16.67%。
百分数值的含义是相对另一个值的比例，另一个值可能是其他属性的值，也可能是继承自父元素的值。
接受百分数值的属性会定义取值范围，以及百分比相对于什么计算。
子元素（一般）继承其父元素计算值，而不是百分数值。

弹性值

弹性值的形式是<number>fr，例如1fr。这个概念由栅格布局引入，用于把布局中不受限制的空间分成几部分。

长度值

长度值用数字后跟一个单位表示，在取指句法中用<length>表示。某些属性只接受正数。例如20px、0.25in。长度值为0时不需要单位。

绝对(物理)长度单位

  • in：英寸。1英寸等于2.54厘米。
  • cm：厘米。1厘米等于0.394英寸。
  • mm：毫米。1厘米等于10毫米。
  • q：四分之一毫米.
  • pt：点。按惯例，1点等于1/72英寸。点是一个标准印刷度量单位，在打印机和打字机上已经使用数十年。
  • pc：派卡(pica)。1派卡等于12点，等于1/6英寸。。派卡也是一个印刷术语。
  • px：像素。1像素等于1/96英寸。实际上CSS中的像素与显示器的物理像素并不一致，1个CSS像素实际用几个物理像素显示取决于设备像素比devicePixelRatio ，例如Win10默认125%放大，设备像素比为1.25，因此100个CSS像素实际使用125个物理像素显示。

相对长度单位

相对长度根据其他值的变化而变化。子元素不会继承为其父元素指定的相对值，而是继承计算值。

  • em：1em等于元素自身的font-size属性值。例如元素的font-size: 14px;则1em就等于14px。在font-size属性上使用该单位，1em等于元素的父元素的font-size属性值。
  • ex：1ex等于元素第一个可用字体中小写字母x的高度。不含小写字母x的字体中也定义了ex。不同字体里ex基本是不同的。在font-size属性上使用该单位，1ex等于元素的父元素的1ex值。如果字体定义了x-height高度，就用它。如果没有，用户代理可以根据一个小写字母的高度确定x-height，例如看小写字母o的字形延伸到基线下方多远，并减去边界框的top值。如果用户代理无法确定x-height，将1ex设定为0.5em。
  • rem：root em。1em等于<html>元素的font-size属性值。
  • ch：1ch等于所用字体中数字0的进宽。进宽指一个字形的起点到下一个字形的起点之间的距离，一般情况下等于字形本身的宽度加上侧边的间距（间距可正可负）。在等宽字体中，所有字符宽度都是1ch。在其他字体中，字符的宽度可能比0宽，也可能比0窄，不能认为是1ch宽。
  • vw：viewport width。1vw等于视口宽度的百分之一。例如视口宽度为888px，则1vw等于8.88px。如果视口宽度发生了变化，vw的实际值也随之变化。
  • vh：viewport height。1vh等于视口高度的百分之一。例如视口高度为500px，则1vw等于5px。如果视口高度发生了变化，vh的实际值也随之变化。
  • vmin：viewport minimum。1vmin=min(1vw, 1vh)。例如视口宽度888px，高度500px。则1vmin等于5px。
  • vmax：viewport maximum。1vax=max(1vw, 1vh)。1vmax等于视口宽度或高度的百分之一，取较大的那个。例如视口宽度888px，高度500px。则1vmax等于8.88px。

分辨率单位

分辨率单位仅能用在媒体查询中，例如@media (min-resolution: 500dpi)。

  • dpi：点每英寸。在长为1英寸的范围内能显示的点数。可以指打印机输出的点，也可以指LED屏幕上的物理像素点，或者电子墨水屏上的图像单元。
  • dpcm：点每厘米。在长为1厘米的范围内能显示的点数。
  • dppx：点每像素单位。每个像素单位显示的点数，在CSS3中，1dppx等于96dpi。

计算值

声明的属性值有时是一个计算值，形式是calc()，在括号中写要进行的计算，支持加+减-乘*除/和括号，例如calc(90% - 2em)。
在允许使用<length>、<frequency>、<angle>、<percentage>、<number>和<integer>的地方都能使用calc()。
对于百分值，它表示使用calc()的属性的在计算calc()之前的值。
calc()还有一些限制：

  1. +和-两侧值必须使用相同的单位类型，或者是<number>和<integer>。例如1 + 3.5是有效的，而1em + 3.5是无效的。但1em + 3.5px是有效的，因为em和px都是长度单位。
  2. *两侧值必须有一个是<number>。例如2.5rem * 2和2 *2.5rem是有效的，结果为5rem。而2.5rem * 2.5rem是无效的。
  3. /右侧值必须是<number>，且不能是0。当左边是<integer>时，结果为<number>值；否则，结果的单位与左边的值一样。例如30em / 2.75是有效的，而30 / 2.75em是无效的。
  4. +和-运算符两侧必须有空白，*和/两侧可以没有空白。

属性值

有时可以使用attr()获取到对应元素上的HTML属性值。
例如p::before {content: "[" attr(id) "]";}在插入文本中使用了元素<p>的id属性值。

颜色值

特殊关键字

  • transparent：表示完全透明的颜色，等价于rgba(0,0,0,0)
  • currentColor：表示元素的color属性值。当把color属性设为currentColor时，它表示继承的color值。

命名颜色

CSS定义了一个大的命名颜色集合，使一些常用颜色易读易写。例如

  • aqua：青色
  • red：红色
  • blue：蓝色

等等。更多命名颜色可以参考named colors

函数式RGB

函数式RGB表示法的格式是rgb(color)，其中color是以逗号分隔的三个整数，或者三个百分数，三个数必须都是百分数或者都是整数，不能混用。整数的取值范围是0-255，百分数的取值范围是0%-100%。超出范围将被裁剪到最接近的边界值。百分数值乘以255等于相应的整数值。例如rgb(75%,50%,50%)或rgb(191,127,127)。

函数式RGBa

函数式RGBa表示法的格式是rgba(color,alpha)，其中alpha值alpha通道，取值范围是0-1之间的实数。完全透明alpha值为0，完全不透明alpha值为1。例如rgba(0,0,0,0.4)

十六进制RGB

十六进制RGB表示法用井号后跟三组两位十六进制数来表示颜色，例如#FF0000或#808080。
CSS支持一种简短写法，即井号后跟三组一位十六进制数来表示颜色。等价于将每组中的数字复制成两个的三组两位十六进制表示法。即#F0A等价于#FF00AA，#6FA等价于#66FFAA。大多数颜色无法用简短写法表示，例如#808080。

十六进制RGBa

十六进制RGB表示法用井号后跟四组两位十六进制数来表示颜色和alpha通道，例如#FF0000CC或#808080AA。
alpha通道的值由十六进制表示，将它转换为十进制后除以255得到0-1之间的实数。

函数式HSL

函数式HSL表示法的格式是hsl(color)，其中color是以逗号分隔的Hue(色相)、Saturation(饱和度)和Lightness(亮度)。色相是角度值，范围是0-360；饱和度和亮度是百分数值，范围是0%-100%。
色相环正上方是0度，角度值顺时针增加。


例如hsl(60,0%,25%)或hsl(180,0%,75%)

函数式HSLa

函数式HSLa表示法的格式是hsla(color,alpha)，其中alpha值alpha通道，取值范围是0-1之间的实数。完全透明alpha值为0，完全不透明alpha值为1。例如rgba(0,0,0,0.4)

特殊关键字

在任何使用颜色的地方都可以使用以下两个特殊关键字之一：

  • transparent：表示透明，等价于``rgba(0,0,0,0)`
  • currentColor：等于元素自身的color属性值。例如main {color: red; border-color: currentColor;}将边框颜色设置成和自身color属性值一样。

角度

角度的形式是<number>后跟以下四种单位之一，在取值句法中用<angle>表示。

  • deg：度数。完整的圆周是360度。
  • rad：弧度。完整的圆周是2pi
  • trun：圈数。完整的圆周是1turn。
  • grad：百分度。完整的圆周是400百分度。

时间

时间的形式是<number>后跟以下两种单位之一，在取值句法中用<time>表示。通常用在过渡和动画中，用于定义持续时间或延迟时间。

  • s：秒。
  • ms：毫秒。1秒等于1000毫秒。

频率

频率的形式是<number>后跟以下两种单位之一，单位标识符不区分大小写，在取值句法中用<frequency>表示。

  • Hz：赫兹。
  • kHz千赫兹。1kHz等于1000Hz。

位置

位置在取值句法中用<position>表示。它的形式是

意思是：

  • 如果只声明一个值，例如left或25%，第二个值将被自动设为center。因此left等价于left center。因此center的意思是就是center center，水平垂直都居中。
  • 如果声明两个值（包括一个值隐式扩展到两个值的情况），而且第一个值是长度或百分数，则该第一个值是横向值。例如25% 35px中25%是距离左边的横向偏移，35px是距离上边的纵向偏移。这意味着25% left或35px right的写法是无效的，因为两个值都是横向值，没有指定纵向距离。但是left 25%或者35px top是可以的，因为第一个值是横向值，第二个值是纵向值。33% 30px等价于left 33% top 30px
  • 如果声明四个值，必须有两个长度或百分数，而且第一和第三个值都得是关键字。此时，长度或百分数指定偏移距离，关键字定义偏移以哪一边计算。right 10px bottom 30px表示右边向左偏移10像素，底边向上偏移30像素。
  • 如果声明了三个值，必须有两个关键字和一个长度或百分数，长度或百分数必须在第二或第三个位置，表示它前面的关键字的偏移值。另一个字的关键字偏移值为0。即right 20px top等价于right 20px top 0。
