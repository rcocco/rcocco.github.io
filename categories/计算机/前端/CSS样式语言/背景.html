背景
2020-11-03|计算机前端CSS样式语言

背景区域是整个 border box，边框会绘制在背景区域之上。

背景颜色

background-color属性用于设置元素的背景颜色。通过有透明背景色的元素可以看到祖先元素的背景色。

  • 取值语法：
  • 初始值：transparent
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 计算值：指定的值
  • 继承性：否
  • 动画性：是

背景图片

background-image属性用于把图片放到背景区域中。设置该属性的同时最好也设置背景颜色，以便在图片无法正常加载能够看清楚文字。同时如果背景图片有alpha通道，即部分区域是透明的，也将能看到背景颜色。当有多个背景时，不会用声明的值重复，你必须声明每一个背景图片。

  • 取值语法：
  • 初始值：none
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 计算值：指定的值，但URL会变成绝对URL
  • 继承性：否
  • 动画性：否

背景定位

background-position属性用于指定图片在背景区域的位置。当有多个背景时，用声明的值重复，例如声明了a b c，如果背景少于等于三个，则取值分别为a、b、c，如果多于三个，则第四个是a，第五个是b，依次重复。

  • 取值语法：
  • 初始值：0% 0%
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 百分数：元素与图像的尺寸的百分数，指定元素和或源图像上用来对齐的点
  • 计算值：指定<length>时是绝对长度偏移，否则是百分数值
  • 继承性：否
  • 动画性：否

属性值可以是一到四个值，如果使用两个非关键字值，则第一个值代表水平位置，第二个值代表垂直位置。如果仅指定一个值，则假定第二个值为center。如果使用三个或四个值，则长度和百分比值是先前关键字的偏移量。

关键字的值就表明了含义，例如top right表示把图像放在元素背景右上角，或者说图像的右上角与背景区域的右上角对齐。

长度值的含义是图片相对于背景区域左上角的偏移，它只作用于背景区域，在背景区域中找到偏移后的点，然后将图片左上角与该点对齐。

1值语法：

  • center：图像的中点与背景区域的中点对齐。
  • top、left、bottom、right：图像的某边与背景区域的某边对齐，另一个方向为center。
  • <length-percentage>：等价于<length-percentage> center

2值语法：

  • 对于关键字，top和bottom表示垂直方向，left和right表示水平方向，谁前谁后是任意的，但必须一个是水平方向，另一个是垂直方向。
  • 对于长度值，表示图片左边和上边相对于背景区域左边和上边的偏移，它只作用于背景区域，在背景区域中找到偏移后的点，然后将图片左上角与该点对齐。类似定位，偏移正值向右向下，负值向左向上。
  • 对于百分数值，它们既要作用于图像来确定图像上的一个点，也要作用于背景区域来确定背景区域中的一个点，然后将两个点重合。水平方向百分数相对于图像或背景区域的左边，偏移(图片/背景区域宽度乘百分数)的距离。垂直方向百分数相对于图像或背景区域的上边，偏移(图片/背景区域高度乘百分数)的距离。类似定位，偏移正值向右向下，负值向左向上。因此50% 50%的含义是，图片中点与背景区域中点重合，即等价于center。0% 0%的含义是，图片左上角的点与背景区域左上角的点重合，即等价于top left。

3值语法：
必须是两个关键字和一个<length-percentage>。<length-percentage>不能是第一个值。
<length-percentage>表示它前面的关键字的偏移量。同样，偏移正值向右向下，负值向左向上。

4值语法：
必须是两个关键字和两个<length-percentage>，<length-percentage>表示它前面的关键字的偏移量。偏移正值向右向下，负值向左向上。

背景图片尺寸

background-size属性用于设定背景图片的尺寸。

  • 取值语法：
  • 初始值：auto auto
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 百分值：相对于背景定位区域background-origin计算
  • 计算值：指定的值，相对长度会转换为绝对长度
  • 继承性：否
  • 动画性：否

可以用以下方式来给定属性值：

  • 只使用关键字cover或contain
  • 仅使用auto或者长度、百分值，它表示宽度。高度值自动为auto
  • 使用两个值，每个值可以是auto、长度或百分值，表示高度。

属性值的具体作用如下：

  • auto：
    • 如果一个方向是auto，另一个方向不是，且图像有宽高比信息，则auto的方向尺寸根据另一个方向的尺寸与宽高比计算。例如宽高比是3比2，给定宽度100，高度auto，则高度被计算为66.67。
    • 如果两个方向都是auto，且图像有尺寸信息，则以该尺寸渲染。
    • 如果图像没有尺寸信息，则采用背景区域的宽度或高度。
  • cover：维持宽高比的情况下缩放图片，如果尺寸不匹配，裁减掉部分以保证没有空白。
  • contain：维持矿高比的情况下缩放图片，如果尺寸不匹配，也不裁剪，而是留空（有重复时重复图片）

背景定位区域

background-origin属性用于改变背景定位区域，它只用于background-image的定位。图片等默认是从padding box的左上角平铺，该属性可以修改这个行为。可以理解为从背景定位区域开始铺图片，然后被background-clip截断。
当background-attachment: fixed时该属性会被忽略。当有多个背景时，用声明的值重复，例如声明了a b c，如果背景少于等于三个，则取值分别为a、b、c，如果多于三个，则第四个是a，第五个是b，依次重复。

  • 取值语法：
  • 初始值：padding-box
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 计算值：指定的值
  • 继承性：否
  • 动画性：否

背景范围

background-clip属性用于控制背景区域的范围。所有背景，包括图片和颜色的作用区域，都取决于这个值。当有多个背景时，用声明的值重复，例如声明了a b c，如果背景少于等于三个，则取值分别为a、b、c，如果多于三个，则第四个是a，第五个是b，依次重复。

  • 取值语法：
  • 初始值：border-box
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 计算值：指定的值
  • 继承性：否
  • 动画性：否

当既有圆角border-radius又有该属性时，背景绘制区域先由background-clip裁剪后，再由圆角进一步裁剪。

背景重复

background-repeat属性用于设定背景图片如何重复。如果图片小于背景区域，默认情况下图片会重复多次以铺满整个背景区域。背景图像可以沿水平和垂直轴重复，也可以完全不重复。默认情况下，重复的图像被裁剪为元素的大小，但是它们可以缩放以适合（使用round）或从一端到另一端均匀分布（使用space）。
重复的起点取决于background-position属性。
当有多个背景时，用声明的值重复，例如声明了a b c，如果背景少于等于三个，则取值分别为a、b、c，如果多于三个，则第四个是a，第五个是b，依次重复。

  • 取值语法：
  • 初始值：repeat
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 计算值：指定的值
  • 继承性：否
  • 动画性：否

属性值的具体作用如下：

  • repeat：图片左上角对齐背景定位区域的左上角（具体而言取决于background-position属性的对齐点），然后开始重复，以便铺满整个背景区域，如果最后的图片尺寸不合适，则会直接裁剪掉。
  • space：图片重复时尽可能不裁剪，第一张重复图片和最后一张重复图片对齐元素的边界，中间如果有空间就放图片，没有空间就放置空白，空白会均匀分布在多张图片之间。当连一张图片都放不下时才会裁剪。除非水平或垂直方向只有一张重复图片，否则background-position属性的对应方向值将被忽略。
  • round：图片会重复，并且会被拉伸。当剩余空间大于等于图片尺寸的一半时，添加下一张重复图片，然后调整所有图片的拉伸以便继续填满整个区域。
  • no-repeat：图片不重复。

该属性可以有两个值，第一个值表示水平重复方式，第二个值表示垂直重复方式。
只有第一个值时，表示两个值的简写，即下表：

单个值 等价的两个值
repeat-x repeat no-repeat
repeat-y no-repeat repeat
repeat repeat repeat
space space space
round round round
no-repeat no-repeat no-repeat

背景粘附

background-attachment属性用于设定背景图片是否在视口内固定，又或者随包含块滚动。当有多个背景时，用声明的值重复，例如声明了a b c，如果背景少于等于三个，则取值分别为a、b、c，如果多于三个，则第四个是a，第五个是b，依次重复。

  • 取值语法：
  • 初始值：scroll
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 计算值：指定的值
  • 继承性：否
  • 动画性：否

属性值的具体作用如下：

  • fixed：背景图片相对于视口固定。当元素内容滚动时，图片不滚动。页面滚动时，图片也不滚动。当然图片仍显示在元素框中，随着页面的滚动，元素框被移动到上面，你将只能看到图片的上半部分。利用这一点可以实现很多效果，例如用多张只有颜色不同的产品图片在不同元素处固定，随着滚动，元素框上移，旧颜色也被滚动上去，形成一种颜色随着水平扫描线变化的效果。
  • local：背景图片相对于元素内容固定，就好像二者是一体的。当元素内容滚动时，图片会滚动。当页面滚动时，图片也会滚动。
  • scroll：背景图片相对于元素本身而不是内容固定。当元素内容滚动时，图片不滚动。当页面滚动时，图片会滚动。
  • round：图片会重复，并且会被拉伸。当剩余空间大于等于图片尺寸的一半时，添加下一张重复图片，然后调整所有图片的拉伸以便继续填满整个区域。
  • no-repeat：图片不重复。

简写属性

background是上述八个背景相关属性的简写属性。

  • 取值语法：
  • 初始值：和单独属性一致
  • 适用于：所有元素，还可以应用到伪元素::first-letter和::first-line上
  • 百分值：和单独属性一致
  • 计算值：和单独属性一致
  • 继承性：否
  • 动画性：和单独属性一致

属性值的顺序没有严格要求，但是有四个规则：

  • 如果有background-size，则必须放在紧跟着background-position的后面，且要用斜线/分隔。
  • 横向值在前，纵向值在后
  • 对于background-origin和background-clip，如果提供两个值，则第一个是background-origin，第二个是background-clip；如果只提供一个值，则该值同时用于这两个属性
  • 没有写出的值也会自动设置为默认值，因此该属性会改变所有上述属性，不会维持属性不变。
  • 可以写多个背景，但要用逗号隔开，且只有最后一个背景可以设置背景颜色background-color

多个背景类似于图层，写在前面的背景的图层，位于写在后面的背景图层之上，因为前面的背景会盖住后面的背景。
