列表项目前面有个记号，例如小圆点或者数字。

列表前面的记号，与后面内容的距离有时是用padding(Chrome)设置的，有时是用margin设置的。

记号类型

list-style-type属性可以改变记号类型

  • 取值语法：
  • 初始值：disc
  • 适用于：列表项目
  • 计算值：指定的值
  • 继承性：是

该属性适用于<li>元素，但由于该属性是继承的，一般直接在<ul>元素上设置即可。它的属性值非常非常多：

属性值none可以使列表项目不生成标记。

列表项目图像

list-style-image属性可以用图像作为记号

  • 取值语法：
  • 初始值：none
  • 适用于：列表项目元素
  • 计算值：none或者绝对URL
  • 继承性：是

列表记号位置

默认情况下，列表记号不在列表项目的内容区中：

通过list-style-position属性可以改变这个行为，使得记号位于内容区中。

  • 取值语法：
  • 初始值：outside
  • 适用于：列表项目元素
  • 计算值：指定的值
  • 继承性：是

属性值具体作用如下：

  • inside：使记号位于主块框（内容区）中。
  • outside：使记号位于内容区外。

简写属性

list-style属性是list-style-image、list-style-position和list-style-type的简写属性。

  • 取值语法：
  • 初始值：disc outside none
  • 适用于：列表项目元素
  • 计算值：指定的值
  • 继承性：是
