生成内容的伪元素

伪元素::before和::after可以生成内容。
当::before和after的content属性值不是none时，这些伪元素将生成框，就好像它们是其原始元素的直接子元素一样，并且可以像在文档树中的元素一样对它们进行样式设置。
::before表示在原始元素实际内容之前的可设置样式的子伪元素。
::after表示在原始元素实际内容之后的可设置样式的子伪元素。
例如p.note::before { content: "Note: " }在每个具有类名note的p元素的内容之前插入文本Note:。由于display的初始值是inline，它会生成一个行内框。就像<p>的其他的行内子元素一样，它会参与<p>的行内格式化上下文，有可能与其他内容共享同一行。

和其他常规元素的内容一样，对于::before和::after伪元素的生成内容，::first-letter和::first-line判断首字母和首行时也需要考虑进去。

content属性所能设置的值非常多，不仅仅是字符串，还可以通过URL设置图片、引号、应用其他属性等等。还可以同时设置多种，只需要用空格分隔。

引号

quotes属性可以为元素生成引号，但仅当::before具有content: open-quote;，::after具有content: close-quote;才有效。就和HTML元素<p>元素的默认设置一样。如果对于某个位置不想生成引号，但又希望下一级引用能使用下一级引号，可以使用content: no-open-quote;或者content: no-close-quote;。

  • 取值语法：
  • 初始值：取决于用户代理
  • 适用于：所有元素，但仅当设置了对应的伪元素才有效。
  • 计算值：指定的值
  • 继承性：是
  • 动画性：否

quotes属性值需要是偶数个以空格分隔的字符串，quotes: "«" "»" "‹" "›";表示第一级引用使用前两个作为开始引号和结束引号，第二级引用使用后两个作为开始引号和结束引号。

计数器

重置或设置

counter-reset将计数器重置为给定的值。
如果一个元素或伪元素上的counter-increment或者content引用了一个不处于任何counter-reset作用域的计数器，实现应该表现得就像已经通过该元素或伪元素上的counter-reset把该计数器重置为0了一样

  • 取值语法：
  • 初始值：none
  • 适用于：所有元素
  • 计算值：指定的值
  • 继承性：否
  • 动画性：否

<custom-ident>用于命名计数器。 <integer>用于设置每次出现元素时要重置成值，如果不指定则是0。
计数器具有作用域，作用域从文档中具有counter-reset该计数器的第一个元素开始，包括该元素的后代、后续兄弟以及后续兄弟的后代。
如果在一个计数器的作用域中又生成了一个同名计数器，则该计数器将创建新的作用域，并且在该作用域内覆盖之前的同名计数器。
