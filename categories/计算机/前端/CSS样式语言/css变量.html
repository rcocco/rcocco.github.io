css变量
2020-11-04|计算机前端CSS样式语言

CSS支持自定义变量。

定义变量

变量以双中划线--开头，例如--name: value;。
变量和属性的声明完全一样，它需要声明在一个选择器的声明块内，不能独立在外部。
类似继承属性，变量也是继承的，并且参与层叠，这意味着变量也有特指度，相同名字的变量，写在选择器#a内的变量会覆盖写在选择器.b中的变量。

使用变量

var( <custom-property-name> , <declaration-value>? )
第一个参数是变量名，如--myvar，第二个可选的参数是备用值，如果第一个参数无效则使用第二个备用值，备用值中除了少数特殊符号外都可以使用，例如var(--foo, red, blue)中，red,blue被认为是一个默认值。
color: var(--myvar)
变量也可以用在calc()函数中，例如calc(var(--myvar) * var(--offset))
