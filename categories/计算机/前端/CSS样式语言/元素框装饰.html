元素框装饰
2020-10-21|计算机前端CSS样式语言

元素盒阴影

box-shadow属性为元素框添加阴影。

  • 取值语法：
  • 初始值：none
  • 适用于：所有元素，还可以应用到伪元素::first-letter
  • 计算值：一个颜色加三个绝对长度
  • 继承性：绝对长度或指定的值
  • 动画性：是

属性值为none时没有阴影。
可以添加一个或多个阴影，彼此之间用逗号分隔。
每个阴影由一个可选的inset、2个长度值后跟2个可选的长度值，以及可选的颜色组成

  • inset表示阴影向盒内扩散，如果不指定该关键字，则默认向盒外扩散。
  • 颜色设定的是阴影的颜色，如果不设置将于文本颜色相同。
  • 前两个长度值是阴影离盒的距离，第一个长度值设定横向偏移，正的长度值向右偏，负的长度值向左偏；第二个长度值设定纵向偏移，正的长度值向下偏，负的长度值向上偏。
  • 可选的第三个长度值定义阴影的模糊半径。模糊半径是从阴影轮廓到模糊效果边界的距离。这个值越大越看不清阴影中的盒子的形状，会扩散成模糊的一片。不允许负值，默认情况为0，即阴影轮廓非常清晰。
  • 可选的第四个长度值是传播半径，正值导致阴影增大，负值导致阴影缩小。默认为0，此时阴影与元素同样大。

轮廓

轮廓和边框类似，一般画在边框外侧，但和边框也有区别：

  1. 轮廓不占空间，它仅仅是视觉效果，不影响布局。
  2. 轮廓可以不是矩形，也不可以不连续。
  3. 通常在元素处于:focus状态时才渲染轮廓
  4. 轮廓只能作为四条边的整体设置，不能为单独一边设置轮廓

轮廓样式

轮廓的样式是指轮廓线条的类型，如单实线、双实线、虚线、雕刻风格线等等。
outline-style属性用于设置轮廓样式。

  • 取值语法：
  • 初始值：none
  • 适用于：所有元素
  • 计算值：指定的值
  • 继承性：否
  • 动画性：否

轮廓和边框样式只有两处不同，其他都完全相同。

  1. 轮廓没有hidden
  2. auto属性表示由用户代理自定义轮廓样式，既可以是其他样式的一种，也可以比其他样式更加华丽。

轮廓宽度

轮廓的宽度是指轮廓线条的粗细。
outline-width用于设置轮廓粗细。

  • 取值语法：
  • 初始值：medium
  • 适用于：所有元素
  • 计算值：当轮廓样式为none时，计算值为0；否则为绝对长度。
  • 继承性：否
  • 动画性：是

属性值效果和边框宽度效果一样。

轮廓颜色

outline-color属性用于指定轮廓颜色。

  • 取值语法：
  • 初始值：currentColor
  • 适用于：所有元素
  • 计算值：指定的值
  • 继承性：否
  • 动画性：是

还有一个invert属性值，表示反转颜色，但除了IE没有任何浏览器支持。

简写属性

outline属性用于设置轮廓，它是outline-style、outline-width和outline-color的简写属性。

  • 取值语法：
  • 初始值：和被简写的属性一样
  • 适用于：所有元素
  • 计算值：和被简写的属性一样
  • 继承性：否
  • 动画性：和被简写的属性一样
