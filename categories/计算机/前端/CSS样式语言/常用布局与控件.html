常用布局与控件
2021-03-22|计算机前端CSS样式语言

https://zh.learnlayout.com/display.html

布局

横向布局

  • 方法1：将需要横向排列的元素设为inline-block，它天然就是横向排列。注意inline-block元素之间会有空格，因为它被视为一个单词，而源代码中的换行符被压缩成空格，消去空格要么将源代码中的空白符删除，要么将父元素字号设为0再在子元素调整回正常字号。
  • 方法2：将需要横向排列的元素设为block，依次调整左外边距使其水平方向上错开，最后利用负的上外边距将下方元素移到同一行。
  • 方法3：定位。通过四个偏移属性调整你想要的任何位置。
  • 方法4：浮动。浮动的元素会尽可能向左或者向右移动，并且后一个浮动元素会挨着前一个浮动元素。
  • 方法5：弹性盒。将父元素设为弹性容器，就可以任意的控制作为子元素的弹性元素的布局。

纵向布局

  • 方法1：将需要纵向排列的元素设为block，它天然就是纵向排列。
  • 方法2：将需要纵向排列的元素设为inline-block，然后使元素的宽度与包含块的宽度一致，从而将下一个inline-block元素挤到下一行。
  • 方法3：定位。通过四个偏移属性调整你想要的任何位置。
  • 方法4：浮动。浮动的元素会尽可能向左或者向右移动，但当没有位置时，它会下移到前一个浮动元素底部，再向左或者向右移动。
  • 方法5：弹性盒。将父元素设为弹性容器，就可以任意的控制作为子元素的弹性元素的布局。

两个元素总宽度固定，各自自动调整宽度，前者宽后者就小，前者小后者就宽。

  • 外层display: table，内层display: table-cell，为前者设置width: 1%，后者设置width: 100%。因为表格布局永远不会超出宽度。

居中

注意，元素居中和元素内文本居中是两个概念，后者的问题实际是匿名行内元素的居中。

水平居中

  • 一个或多个行内元素：为它的块容器父元素设置text-align: center
  • 一个块级元素：设置margin-left:auto和margin-right:auto，前提是宽度、横向边框宽度、横向内边距加起来小于等于包含块宽度，且border-left-width + padding-left等于border-right-width + padding-right。非替换元素还要求width不是auto，替换元素不对width作要求。
  • 一个或多个任意元素：将父元素变成弹性容器，然后设置justify-content: center。
  • 一个任意元素：将父元素变成弹性容器，然后设置margin-left: auto和margin-right:auto吸收多余空白。
  • 一个绝对定位元素：
    • 设置margin-left:auto和margin-right:auto，前提是left和right不是auto，且左右偏移、横向边框宽度、横向内边距加起来小于等于包含块宽度，且left + border-left-width + padding-left等于border-right-width + padding-right + right。非替换元素还要求width不是auto（为了避免滚动条我们一般把max-width设为非none），替换元素不对width作要求。
    • 设置left:50%，此时元素左边界位于包含块水平中线，再通过平移translateX(-50%)将元素的水平中线向左移动元素自身宽度的50%。

垂直居中

  • 一个或多个高度auto的行内元素：为父元素添加伪元素::before，使伪元素display: inline-block以便在维持行内格式化的情况下设置高度，令高度为父元素高度的100%，并设置vertical-align: middle，令它的垂直中线（父元素垂直中线）与 stuct 盒的x的中心对齐。然后令高度auto的行内元素vertical-align: middle，使得它的盒的垂直中线与stuct盒的x中心对齐，也就是与父元素垂直中线对齐了。
  • 一个绝对定位元素：
    • 设置margin-top:auto和margin-bottom:auto，前提是top和bottom不是auto，且上下偏移、纵向边框宽度、纵向内边距加起来小于等于包含块高度，且top + border-top-width + padding-top等于border-bottom-width + padding-bottom + bottom。非替换元素还要求height不是auto，替换元素不对height作要求。
    • 设置top:50%，此时元素上边界位于包含块垂直中线，再通过平移translateY(-50%)将元素的垂直中线向上移动元素自身高度的50%。
  • 一个或多个只有单行文本的元素：设置line-height等于容器高度。
  • 一个或多个单元格：将元素设为display: table-cell然后使用vertical-aign: middle。根据vertical-align应用于单元格时的规定，单元格的中心与它跨越的行的中心对齐。
  • 一个或多个任意元素：将父元素变成弹性容器，然后设置align-content: center。
  • 一个任意元素：将父元素变成弹性容器，然后设置margin-top: auto和margin-bottom:auto吸收多余空白。
  • 方法4：定位+2D变换

特定bug解决方案

页面顶部有空隙

body元素已设置margin: 0顶部仍然有空隙，html等元素无内外边距。
原因是外边距折叠，body内部第一个有外边距的元素的外边距溢出了。

垂直换行flex宽度不正确

对于一个垂直换行、宽度auto的flex容器，当出现换行后浏览器在计算宽度时会出现问题，不管有几列，浏览器永远只计算第一列的宽度，这是一个bug，与规范要求不符。例如：

由于浏览器对写入模式的支持非常不错，一个解决办法时，将flex容器的写入模式改为垂直写入，此时的flew-direction:row就是平常横向写入模式的flew-direction:column，然后将弹性元素的写入模式改回水平写入。

轮播图

纯CSS：利用input:radio单选按钮元素，配合:checked伪类使得图片元素从透明到不透明，或者从定位右到定位左。注意，组合选择器只能选择后代、子代、后面的兄弟元素，而input:radio是自闭合标签没有后代子代，因此图片元素是input:radio的后面的兄弟元素，或者后面的兄弟元素的后代。另外通过<label>元素，可以将其与单选按钮关联，从而可以给图片两侧点击按钮，一个<label>只能关联一个<input>，但多个<label>可以关联同一个<input>。如果不想两者是兄弟或者兄弟的后代，只能用定位移到特定位置。

百叶窗动画轮播图

根据图片大小分成四个div，第一个div里有n张图片，每张图片是左侧四分之一，第二个div里有n张图片，每张图片是四分之一，以此类推。为了代码简洁以及兼容性，图片最好使用span中背景图片的形式，而不是直接放图片<img>。
没有选中的图片定位到left: -100%，选中的图片定位到left: 0。
为了实现更好的效果，在选中的图片移动时，没选中的图片也应该同向移动，使用动画left: 0到left: 100%可以实现。

菜单

<ul>与<ul>嵌套，一级菜单inline-block横向排列，二级和更子级菜单平时display: none不显示，仅在父<li>悬停时display:block纵向排列；二级菜单绝对定位于top: 100%; left:0；三级和之后的菜单绝对定位于top: 0; left: 100%。由于子元素宽度不超过包含块宽度，为了让菜单宽度自适应最长的菜单项，使用white-space:nowrap阻止换行。

维持hover状态

利用过渡可以使得鼠标不再hover元素后也可以维持hover状态。

  • 一组兄弟元素，维持其中一个的hover状态：

没有hover时过渡延迟无穷大，hover父元素时取消子元素过渡并回到不hover时的属性，hover元素时取消过渡并改变属性。
这个方法的缺点是当子元素没有完全占满父元素时，hover父元素的空隙会导致正在维持的hover状态被取消。

若父元素有边框，为了不点击父区域，可以负margin。
或者把父元素宽高设为0再flex等办法。

正文少页脚也在最下面

效果：
正文多时页脚在正文下面，就和默认行为一样。
正文少时，页脚在第一页最下边，而不是紧跟着正文，和默认行为不同。

方法：
为html设置100%，为body设置min-height: 100%并设置一些下内边距。页脚绝对定位在body底部。

原理：
html的高度百分数是相对于初始包含块的，100%就是第一页视口高度。body的高度取决于子元素的高度，又由于min-height: 100%，当body子元素的高度之和小于包含块html元素的高度时，就会是100%，即视口高度。

创建空间立方体

立方体有6个面，需要6个元素。
每个面最开始都是xOy平面，通过3d旋转，使各个平面的Z轴朝向对应方向。然后每个平面都向Z轴方向平移相同的距离。平移距离乘以2就是立方体的边长了。
最后在空间中旋转父元素就可以看到效果。

元素作圆周运动

  1. 元素自身，旋转、X轴移动、反向旋转，其中圆周半径是X轴移动值。
  1. 作为子元素随父元素转动，但子元素自身同时反方向转动使得子元素保持原有方向。圆周半径是父元素的变形中心到子元素中心。

使元素高度与宽度一致

在flex布局中，元素在主轴上的尺寸是可以随可用空间的增加而增加的，但交叉轴不行，要想使交叉轴始终和宽度相同，可以利用伪元素的padding-bottom: 100%，内边距的百分数是相对于包含块宽度的，而元素自身的高度又取决于伪元素的高度。
如果想加子元素可以绝对定位。

又例如元素自身宽度auto，等于包含块宽度，则自身可以设置padding-bottom: 100%使其等于包含块宽度，构成正方形

3d翻转看背面信息

正面背面定位到相同位置。
将背面翻转180度，使得Z轴朝向背面。同时设置backface-visibility: hidden隐藏背面。

生成钟表

div>span{$}*12，原理是先将所有数字定位到钟表父元素正中间，然后旋转使得X轴指向它们应该在的位置，接着平移使它到达应该在的位置，最后反向旋转使元素X轴指回右侧，元素回到正方向。

如果是刻度，通过div>span*60生成60个刻度，然后用同样的方法
