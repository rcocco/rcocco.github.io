嵌套量词

在考虑嵌套量词时，可以当做循环来理解。
另外，除非量词相同，否则量词的顺序很重要，颠倒顺序影响含义。


  1. 如果所有都有为真，则语句为真。
    存在一组使得为假，则语句为假。

  1. 对每个，都存在一个，使得为真，则语句为真。
    有任何一个找不到使为真的，则语句为假。

  1. 存在一个，使得对每个都有，则语句为真。
    如果不存在这样的，则语句为假。

  1. 存在一组使得为真，则语句为真。
    如果所有都有为假，则语句为假。

充分性和必要性

区分充分性和必要性：
从条件出发推出结论是充分性，反之是必要性。
1、证明是的充要条件：
是条件，是结论，充分性是从，必要性是从
2、证明的充要条件是：
是条件，是结论，充分性从，必要性从
3、A当且仅当BA的充要条件是BB是A的充要条件

区分A是B的充分条件与A的充分条件是B：
A是B的儿子，A的儿子是B，问谁是儿子？

A是B的充分条件，A是B的必要条件

P当且仅当Q(P,if and only if)
P当Q(P if Q)：当Q成立时，P一定成立。即有，P是Q的必要条件。
我打伞 if 雨天：当雨天时，我一定打伞。
P仅当Q(P only if Q)：仅当（只有）Q成立时，P（才）成立。即当Q不成立时，P也不成立,。等价逆否命题为：当P成立时，Q才成立。P是Q的充分条件。
我打伞 only if 雨天：仅当雨天时，我才打伞。即如果不是雨天，我一定不打伞。等价逆否命题为：当我打伞时，一定是雨天。
