时间和日期

ECMAScript中的日期对象Date在内部使用Number类型表示时间值。
时间值的范围比Number所能精确表示的整数范围略小，为-8,640,000,000,000,000到8,640,000,000,000,000毫秒。规定UTC时间1970年1月1日午夜的起始为+0。负的毫秒数表示的时间相对于UTC 1970年1月1日午夜往前推，正的毫秒数往后推。因此能表示的时间是相对于1970年1月1日的-100,000,000天到100,000,000天，或-273,790到273,790年。

获取字符串

解析日期时间字符串

合法的日期时间字符串有下面三种形式

Date.parse( str )

返回Number类型。
首先使用抽象操作ToString(str)将参数转为字符串。
然后尝试按照指定格式解析该时间字符串，如果能解析则返回用Number类型表示的时间值。
如果不能解析则返回NaN。

new Date ( str )

返回Date类型对象。
按照指定格式解析该时间字符串，返回的Date对象表示解析出的时间值，如果无法解析会返回表示无效时间的Date对象。

手动设置日期时间

所有参数可以超出范围，例如分钟的有效数字是00到59，但可以超出范围例如-1或60，负值往前推时间，正值往后推时间。
当year在0到99之间时，令year变成year + 1900，因此公元0到99年无法通过传入year来直接表示，必须通过超出范围的其他参数得到。

基于当地时间

返回表示该时间的Date类型对象。

基于UTC时间

返回表示时间的Number值，NaN表示无效或超出范围的时间。

基于Number设置时间

返回对应于该number时间值的Date类型对象。

Date对象

完整年份

返回一个表示完整年份的整数，如果Date对象表示无效时间则返回NaN

月份

返回0到11之间的整数，0表示一月，1表示二月，11表示十二月，如果Date对象表示无效时间则返回NaN

日期

返回1到31之间的整数，如果Date对象表示无效时间则返回NaN

改变日期，可以超出1到31，表示更往前或更往后。

星期几

返回0到6之间的整数，0表示周日，1表示周一，6表示周六。
如果Date对象表示无效时间则返回NaN

小时

返回0到23之间的整数，如果Date对象表示无效时间则返回NaN

分钟

返回0到59之间的整数，如果Date对象表示无效时间则返回NaN

秒数

返回0到59之间的整数，如果Date对象表示无效时间则返回NaN

毫秒

返回000到999之间的整数，如果Date对象表示无效时间则返回NaN

内部Number时间值

返回一个表示时间的Number值，如果Date对象表示无效时间则返回NaN

本地时区偏移

返回-720到720之间的整数，它是以分钟表示的本地时区偏移，即从本地时区加上这个分钟数就得到了UTC时区。具体而言，东时区负，西时区正，简称东负西正。
该方法实际上与Date对象内部有效时间值无关，只要是有效的，不管是什么时间，总会返回相同的结果。如果Date对象表示无效时间则返回NaN。

拷贝Date对象

返回一个新的Date对象，其时间值与参数对象相同

TypeScript 2021-03-30|计算机编程语言JavaScript语言

TypeScript是JavaScript的超集，任何合法的JavaScript代码都是合法的TypeScript代码。
TypeScript的目的是为JavaScript提供静态类型检查，使得JavaScript更加清晰明确。
你可以将TypeScript当成静态类型版本的JavaScript，它的独有语法类似于Java和C#。
TypeScript的代码无法直接运行，必须要编译为JavaScript，编译通过后TypeScript中的独有语法例如类型声明等都会消失，因此TypeScript无法检查代码运行时的任何问题。
