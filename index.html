DOM概述

DOM指文档（特别是HTML和XML文档）在内存中的表示形式。有时DOM也指用于访问和操作文档的API。
具体而言，每个文档都被表示为一个结点树。树中的一些结点可以有子结点，而另一些结点则始终是叶子结点。

Element结点

Element结点简称元素。
元素有关联的namespace、namespace prefix、local name、custom element state、custom element definition、is value。元素创建时，所有这些关联属性都被初始化。
custom element state为uncustomized或custom的元素称为定义的。custom element state为custom的元素称为自定义的。
元素有一个关联的影子根，除非另有说明否则为null，当影子根非null时元素被称为影子宿主。

Document结点 2021-06-15|计算机前端DOM

Document结点

Document结点简称为文档。
每个文档都有关联的编码、内容类型(content type)、URL、origin、类型、模式。
一个类型为xml的文档称为XML文档，类型为html的文档称为HTML文档。
模式为no-quirks的文档称为非怪异模式的，模式为quirks的文档称为怪异模式的，模式为limited-quirks称为有限怪异模式。
非怪异模式最初被称为"标准模式"，有限怪异模式曾被称为"准标准模式"。由于它们的细节现在由标准定义，因此被重新命名了（同时因为 Ian Hickson 否决了它们的原名，理由是它们是无意义的）

数组 2021-06-14|计算机编程语言JavaScript语言

概述

在JavaScript中，数组是一个具有"length"属性，以及一组属性键为数组索引的可枚举属性的对象。

“整数索引”是用作属性键的字符串，它是规范的数字字符串，即满足ToString(ToNumber(P)) === P的数字字符串，其数字值为[+0, 2^53-1]之间的整数。
“数组索引”是整数索引，其数字值为[+0, 2^32-2]之间的整数，即满足ToString(ToUint32(P)) === P且ToUint32(P)不等于2^32-1。
