字符串 2021-04-04|计算机编程语言JavaScript语言

在JavaScript中，字符串由若干UTF-16代码单元组成。在Unicode中，代码点是0x0000到0x10FFFF之间数字，这些数字大多对应于某个字符，也有的数字目前没有对应任何字符。参见Unicode字符百科。UTF-16代码单元是对Unicode代码点进一步编码得到的一组16 bit数，一个Unicode代码点会被编码为1到2个UTF-16代码单元，代码点0x0000到0xFFFF被编码为1个UTF-16代码单元，代码点0x10000到0x10FFFF被编码为2个UTF-16代码单元。

正则表达式 2021-04-04|计算机编程语言JavaScript语言

正则表达式字面值形如：/pattern/flags
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions
flags是正则标志，它是以下六个字符的任意组合，但每个字符只能出现0次或1次。

  • g：全局模式，表示查找字符串的全部内容，而不是找到第一个匹配的内容就结束。
  • i：不区分大小写，表示在查找匹配时忽略 pattern 和字符串的大小写。
  • m：多行模式，表示查找到一行文本末尾时会继续查找。
  • y：粘附模式，表示只查找从 lastIndex 开始及之后的字符串。
  • u：Unicode 模式，启用 Unicode 匹配。
  • s：dotAll 模式，表示元字符.匹配任何字符（包括\n 或\r）。

元字符( [ { \ ^ $ | ) ] } ? * + .必须转义

单次匹配

如果正则表达式对象没有g和y标志，则从字符串str的索引0处开始尝试匹配。
如果正则表达式对象有g或y标志，则从字符串str的索引this.lastIndex处开始尝试匹配（如果 this.lastIndex < 0则从0处开始）。

如果没有匹配到任何内容（包括开始索引大于字符串长度的情况），则返回null，同时若正则表达式有g或y标志，则将this.lastIndex重置为0，如果没有这两个标志则不重置。

如果匹配到了内容，且正则表达式具有g或y标志，则将this.lastIndex更新为本次匹配处理完的最后一个输入字符的下一个位置的索引。
如果匹配到了内容，则返回一个数组，数组具有以下内容：

  • 数组长度为正则表达式中捕获组的数量加1，也就是用于捕获的左小括号的数量加1
  • 第0个元素是本次匹配到的完整子串，也就是从开始索引到本次匹配处理完的最后一个输入字符的索引
  • 第1到第n个元素是每个捕获组捕获到的子串
  • 数组具有index属性，它是本次匹配的开始索引
  • 数组具有input属性，它是传递进来的参数str
  • 数组具有groups属性，如果没有命名捕获组，则它是undefined，如果有命名捕获组，则它是一个对象，其属性键是命名捕获组的名字，属性值是命名捕获组捕获到的子串

多次匹配

每次匹配返回子串

如果没有g标志，返回this.exec(str)
如果有g标志，则重置this.lastIndex，然后创建一个数组A，之后多次调用this.exec(str)，将每次得到的结果数组的第0个元素添加到数组A里。如果第一次就匹配失败了，则返回null，否则返回数组A。
由于数组A的元素是每次调用exec得到的结果的第0个元素，即每次匹配完整子串，因此捕获组的内容都无法从该方法中得到。

ECMAScript中的日期对象Date在内部使用Number类型表示时间值。
时间值的范围比Number所能精确表示的整数范围略小，为-8,640,000,000,000,000到8,640,000,000,000,000毫秒。
规定UTC时间1970年1月1日午夜的起始为+0。负的毫秒数表示的时间相对于UTC 1970年1月1日午夜往前推，正的毫秒数往后推。因此能表示的时间是相对于1970年1月1日的-100,000,000天到100,000,000天，或-273,790到273,790年。
