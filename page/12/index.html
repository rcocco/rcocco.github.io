基本元素

  • <svg> 包裹并定义整个矢量图。<svg> 标签之于矢量图就如同 <html> 标签之于一个 web 页面。
  • <line> 创建一条直线。
  • <polyline> 创建折线。
  • <rect> 创建矩形。
  • <ellipse>创建圆和椭圆。
  • <polygon>创建多边形。
  • <path>通过指定点以及点和点之间的线来创建任意形状。
  • <defs>定义一个可复用的图形。初始情况下<defs>里面的内容是不可见的。<defs>标签之于矢量图就如同<head>标签之于一个 web 页面。
  • <g>将多种形状组合起来。将组合后的形状置于<defs>中可以让它能够被复用。
  • <symbol>类似于一个组合，但是拥有一些额外的特性。通常被置于 <defs>标签中便于复用。
  • <use>获取在<defs>中定义的复用对象并在 SVG 中显示出来。

模拟信号把设备里读出来的物理量直接使用，
现代计算机存储和处理的信息以二值信号表示。二进制数字(或称"位(bit)")形成了数字革命的基础。对于有10个手指的人类来说，使用十进制表示法是很自然的事情。但是当构造存储和处理信息的机器时，二进制工作得更好。二值信号能够很容易地被表示、存储和运输，例如，可以表示为穿孔卡片上有洞和无洞、导线上的高电压和低电压（即便不精确5V,5.1V,4.9V都可以认为是高电压，非常灵活），或者顺时针或逆时针的磁场。对二值信号进行存储和执行计算的电子电路非常简单和可靠，制造商能够在一个单独的硅片上集成数百万甚至数十亿个这样的电路。

视觉格式化模型 2020-11-06|计算机前端CSS样式语言

CSS中，视觉格式化模型(Visual Formatting Model)描述了用户代理如何处理文档树，以及如何在视觉媒体上显示。
在视觉格式化模型中，文档树中的每个元素根据盒模型生成零个或更多盒子（矩形框），这些盒子的布局受以下因素控制：

  • 盒子的尺寸和类型
  • 定位模式（常规流、浮动和绝对定位）
  • 文档树中元素之间的关系
  • 外部信息（例如视口大小、图片的固有尺寸等等）

CSS支持自定义变量。

定义变量

变量以双中划线--开头，例如--name: value;。
变量和属性的声明完全一样，它需要声明在一个选择器的声明块内，不能独立在外部。
类似继承属性，变量也是继承的，并且参与层叠，这意味着变量也有特指度，相同名字的变量，写在选择器#a内的变量会覆盖写在选择器.b中的变量。
