字符串的组成

在JavaScript中，字符串由若干UTF-16代码单元组成。在Unicode中，代码点是0x0000到0x10FFFF之间数字，这些数字大多对应于某个字符，也有的数字目前没有对应任何字符。参见Unicode字符百科。UTF-16代码单元是对Unicode代码点进一步编码得到的一组16 bit数，一个Unicode代码点会被编码为1到2个UTF-16代码单元，代码点0x0000到0xFFFF被编码为1个UTF-16代码单元，代码点0x10000到0x10FFFF被编码为2个UTF-16代码单元。

23.1.1 Map构造函数

23.1.1.2 AddEntriesFromIterable ( target, iterable, adder )

抽象操作AddEntriesFromIterable接受一个target对象，一个entries的iterable，和一个用来调用的adder函数，target是receiver。

  1. 如果IsCallable(adder)为false，抛出 TypeError 异常
  2. 断言：iterable存在，且不是undefined也不是null
  3. 令iteratorRecord为? GetIterator(iterable)
  4. 重复：
    1. 令next为? IteratorStep(iteratorRecord)
    2. 如果next为false，返回target
    3. 令nextItem为? IteratorValue(next)
    4. 如果Type(nextItem)不是Object，则：
      1. 令error为ThrowCompletion(新创建的TypeError对象)
      2. 返回? IteratorClose(iteratorRecord, error)
    5. 令k为Get(nextItem, "0")
    6. 如果k是非常规完结，返回? IteratorClose(iteratorRecord, k)
    7. 令v为Get(nextItem, "1")
    8. 如果v是非常规完结，返回? IteratorClose(iteratorRecord, v)
    9. 令status为Call(adder, target, « k.[[Value]], v.[[Value]] »)
    10. 如果status是非常规完结，返回? IteratorClose(iteratorRecord, status)

注：参数iterable应该是一个实现@@iterator方法的对象，该方法返回一个迭代器对象，该迭代器产生一个类似于数组的双元素对象，其第一个元素是一个将被用作Map键的值，其第二个元素是与该键对应的值。
